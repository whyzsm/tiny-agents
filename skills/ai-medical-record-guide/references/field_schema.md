# 电子病历字段 Schema

> 本文档定义 AI 智能病历系统的全量字段规范。每个字段包含名称、数据类型、是否必填、临床意义及校验规则。
> **此文档可被替换或扩展以适配不同医疗机构的病历规范。**

---

## 一、通用基础 — 基本信息

| 字段名 | 数据类型 | 必填 | 临床/产品意义 | 校验规则 |
|--------|---------|------|-------------|---------|
| 姓名 | string | ✅ | 患者唯一标识 | 非空，2-20 字符 |
| 性别 | enum(男/女) | ✅ | 用药及风险分层基础 | 仅限"男""女" |
| 年龄 | string | ✅ | 剂量计算、风险评估 | 格式：数字+单位（岁/月/天），如"65岁""3月" |
| 体重(kg) | float | ⚠️ 条件必填 | 化疗药、胰岛素等药量计算 | >0 且 <500；涉及剂量计算时必填 |
| 身高(cm) | float | ⚠️ 条件必填 | BMI 计算 | >0 且 <300；有体重时联动必填 |
| BMI | float | 🔄 自动计算 | 营养评估、肥胖分级 | = 体重 / (身高/100)²，保留1位小数 |
| 科室 | string | ✅ | 归属管理 | 非空 |
| 病区 | string | ⚠️ 住院必填 | 护理单元定位 | 住院场景必填 |
| 床号 | string | ⚠️ 住院必填 | 物理定位 | 住院场景必填 |
| 住院号/门诊号 | string | ✅ | 唯一业务标识 | 非空 |
| 联系人 | string | ⚠️ 推荐 | 紧急联系 | — |
| 联系电话 | string | ⚠️ 推荐 | 随访触达 | 手机号格式校验 |

## 二、门诊场景 — 病史记录

| 字段名 | 数据类型 | 必填 | 临床/产品意义 | 校验规则 |
|--------|---------|------|-------------|---------|
| 主诉 | string | ✅ | 就诊核心原因，RAG 检索锚点 | 非空；建议格式"症状+持续时间"，如"反复头痛3天" |
| 现病史 | text | ✅ | 本次疾病演变过程 | 非空；应包含起病时间、症状演变、诊治经过 |
| 既往史 | text | ✅ | 基础疾病背景 | 非空；无特殊记"既往体健"或"否认特殊病史" |
| 过敏史 | text | ✅🔴 | **极重要**：药物/食物/接触过敏 | 非空；无过敏记"否认药物及食物过敏史"；有过敏必须明确过敏原及反应类型 |
| 家族史 | text | ✅ | 遗传病风险筛查 | 非空；无特殊记"否认家族遗传病史" |
| 个人史 | text | ⚠️ 推荐 | 生活习惯、职业暴露 | — |
| 婚育史 | text | ⚠️ 条件推荐 | 妇产科相关 | 女性患者推荐填写 |
| 月经史 | text | ⚠️ 条件推荐 | 妇产科相关 | 女性患者推荐填写 |

## 三、门诊场景 — 体格检查

| 字段名 | 数据类型 | 必填 | 临床/产品意义 | 校验规则 |
|--------|---------|------|-------------|---------|
| 体温(℃) | float | ✅ | 发热预警 | 35.0-42.0；≥38.0 标记发热 |
| 脉搏(次/分) | int | ✅ | 心律评估 | 40-200；<60 或 >100 标记异常 |
| 呼吸(次/分) | int | ✅ | 呼吸功能评估 | 8-60；<12 或 >20 标记异常 |
| 血压(mmHg) | string | ✅ | 高血压/低血压预警 | 格式"收缩压/舒张压"，如"120/80"；收缩压 60-300，舒张压 30-200 |
| 专科体查 | text | ✅ | 针对性体格检查描述 | 非空 |
| 神志 | enum | ⚠️ 推荐 | 意识状态评估 | 清楚/嗜睡/昏睡/浅昏迷/深昏迷 |
| 一般情况 | text | ⚠️ 推荐 | 总体生命体征描述 | — |

## 四、门诊场景 — 门诊处方

| 字段名 | 数据类型 | 必填 | 临床/产品意义 | 校验规则 |
|--------|---------|------|-------------|---------|
| 处方类型 | enum | ✅ | 处方分类 | 西药/中成药/中草药/检查/检验/处置/材料 |
| 临床诊断 | string | ✅ | 处方依据 | 非空，应与主诉/病史匹配 |
| 药品名称 | string | ✅(药品类) | 处方核心 | 通用名优先 |
| 药品规格 | string | ✅(药品类) | 剂量换算基础 | — |
| 单次剂量 | string | ✅(药品类) | 每次给药量 | 数值+单位，如"0.5g" |
| 给药途径 | enum | ✅(药品类) | 给药方式 | 口服/静脉注射/肌肉注射/皮下注射/外用/吸入/舌下含服 等 |
| 用药频次 | string | ✅(药品类) | 用药间隔 | 标准缩写：qd/bid/tid/qid/q8h/prn 等 |
| 用药天数 | int | ✅(药品类) | 疗程控制 | >0 |
| 总量 | string | ✅(药品类) | 发药依据 | 自动计算：单次剂量 × 频次 × 天数 |
| 检查/检验项目 | string | ✅(非药类) | 辅助诊断 | 非空 |
| 检查部位 | string | ⚠️ 条件必填 | 影像类必填 | 影像学检查必填 |
| 处置内容 | string | ✅(处置类) | 操作描述 | 非空 |

## 五、住院阶段：入院

### 5.1 入院记录

| 字段名 | 数据类型 | 必填 | 临床/产品意义 | 校验规则 |
|--------|---------|------|-------------|---------|
| 病史陈述人 | string | ✅ | 主体识别（患者本人/家属/其他） | 非空 |
| 陈述可靠程度 | enum | ✅ | 信息质量评估 | 可靠/基本可靠/不可靠 |
| 入院途径 | enum | ✅ | 入院方式 | 急诊/门诊/转科/转院/其他 |
| 入院时间 | datetime | ✅ | 时间戳 | ISO 格式或"YYYY-MM-DD HH:mm" |
| 入院诊断 | text | ✅ | 初始诊断列表 | 非空；支持多诊断，以序号分行 |
| 主诉 | string | ✅ | 同门诊 | 同门诊校验 |
| 现病史 | text | ✅ | 同门诊（住院更详细） | 同门诊校验 |
| 既往史 | text | ✅ | 同门诊 | 同门诊校验 |
| 过敏史 | text | ✅🔴 | 同门诊 | 同门诊校验 |
| 家族史 | text | ✅ | 同门诊 | 同门诊校验 |
| 个人史 | text | ✅ | 住院阶段必填 | 非空 |
| 体格检查 | text | ✅ | 入院时全面体查 | 非空；应包含生命体征和专科体查 |
| 辅助检查 | text | ⚠️ 推荐 | 入院前已有检查结果 | — |

### 5.2 风险评估

| 字段名 | 数据类型 | 必填 | 临床/产品意义 | 校验规则 |
|--------|---------|------|-------------|---------|
| 压疮风险评估(Braden) | object | ✅ | 压疮预防 | 评分 6-23；≤18 为高风险 |
| 跌倒风险评估(Morse) | object | ✅ | 跌倒预防 | 评分 0-125；≥45 为高风险 |
| VTE风险评估(Caprini) | object | ✅ | 血栓预防 | 评分 0-n；≥5 为高风险 |
| 营养风险评估(NRS2002) | object | ✅ | 营养干预依据 | 评分 0-7；≥3 需营养支持 |
| 疼痛评估(NRS) | object | ⚠️ 推荐 | 疼痛管理 | 评分 0-10 |
| 自理能力评估(Barthel) | object | ⚠️ 推荐 | 护理等级依据 | 评分 0-100 |
