### 模块二：疾病认知与阶段

#### 什么是阿尔茨海默症？

**疾病本质**：
- 进行性神经退行性疾病
- 大脑神经细胞逐渐死亡
- 记忆力、认知能力、生活能力逐渐下降
- 目前无法治愈，但可以延缓进展

**主要病理特征**：
- 脑内淀粉样蛋白沉积
- 神经纤维缠结
- 脑细胞死亡

#### 疾病阶段

| 阶段 | 持续时间 | 主要表现 |
|------|---------|---------|
| **轻度（早期）** | 2-4年 | 记忆力减退，尤其是近期记忆；语言表达困难；定向障碍；判断力下降；仍能独立生活 |
| **中度（中期）** | 2-10年 | 近期和远期记忆严重受损；语言障碍明显；需要协助日常生活；可能出现行为问题；定向障碍严重 |
| **重度（晚期）** | 1-3年 | 完全依赖照护；丧失语言能力；丧失行动能力；吞咽困难；大小便失禁；并发症风险高 |

#### 常见症状

**认知症状**：
- 记忆力减退（尤其是近期记忆）
- 语言障碍（找词困难、理解困难）
- 定向障碍（时间、地点、人物）
- 判断力下降
- 抽象思维困难

**精神行为症状（BPSD）**：
- 抑郁、焦虑
- 淡漠、退缩
- 激越、攻击
- 妄想、幻觉
- 游走、重复行为
- 睡眠障碍

**功能症状**：
- 日常生活能力下降
- 工具性日常生活能力先受损
- 基本日常生活能力后受损

### 模块三：日常照护核心

#### 3.1 日常生活照护

**饮食照护**：

| 方面 | 照护要点 |
|------|---------|
| **营养** | 均衡饮食，保证蛋白质、维生素 |
| **安全** | 防止呛咳、误吸；食物切小块 |
| **协助** | 中期需要协助进食；晚期需要喂食 |
| **环境** | 安静、熟悉的环境；固定座位 |
| **习惯** | 固定用餐时间；熟悉的餐具 |

**穿衣照护**：
- 选择简单、易穿脱的衣物
- 按顺序摆放衣物
- 给予充足时间
- 尊重患者偏好
- 注意保暖

**洗浴照护**：
- 固定时间（建立习惯）
- 注意水温（防止烫伤）
- 防滑措施
- 保护隐私
- 尊重患者意愿

**如厕照护**：
- 定时提醒如厕
- 观察排便规律
- 中期使用便盆、尿壶
- 晚期使用纸尿裤
- 注意皮肤护理

**睡眠照护**：
- 固定作息时间
- 白天适当活动
- 睡前避免刺激
- 夜间安全措施
- 处理夜间游走

#### 3.2 认知训练与刺激

**记忆训练**：
- 回忆过去（远期记忆保留较好）
- 看老照片
- 听熟悉的音乐
- 讲过去的故事

**感官刺激**：
- 视觉：看照片、看风景
- 听觉：听音乐、听广播
- 触觉：触摸不同材质
- 嗅觉：闻熟悉的气味
- 味觉：品尝熟悉的食物

**活动参与**：
- 简单家务
- 园艺活动
- 手工活动
- 散步
- 社交活动

**注意事项**：
- 活动要简单、安全
- 避免失败体验
- 关注过程而非结果
- 尊重患者意愿

#### 3.3 药物管理

**管理原则**（遵医嘱）：
- 按时给药
- 监督服药（防止漏服、多服）
- 观察药物反应
- 定期复诊

**常见药物类别**：
- 胆碱酯酶抑制剂
- NMDA受体拮抗剂
- 对症治疗药物

**⚠️ 注意**：具体用药请遵医嘱，不可自行调整。

### 模块五：沟通技巧

#### 5.1 沟通原则

| 原则 | 说明 |
|------|------|
| **简单** | 使用简单、短句 |
| **清晰** | 语速慢，吐字清晰 |
| **耐心** | 给予充足时间回应 |
| **尊重** | 尊重患者感受，不否定 |
| **非语言** | 眼神接触、微笑、触摸 |

#### 5.2 有效沟通技巧

**要做的**：
- 用患者喜欢的称呼
- 一次只问一个问题
- 用"是/否"问题
- 用视觉提示
- 重复和确认
- 保持眼神接触
- 微笑、触摸

**避免做的**：
- 大声喊叫
- 使用复杂句子
- 同时给多个指令
- 争论、纠正
- 表现出不耐烦
- 使用婴儿语

#### 5.3 处理重复问题

**原因**：
- 记忆障碍
- 焦虑、不安
- 寻求安全感

**应对**：
- 耐心回答
- 转移注意力
- 提供安慰
- 不要争论

#### 5.4 处理妄想和幻觉

**不要**：
- 争论、否定
- 说"没有的事"
- 表现出不耐烦

**要做**：
- 倾听、安慰
- 转移注意力
- 检查是否有真实原因（如疼痛、感染）
- 必要时就医

### 模块六：行为管理

#### 6.1 常见行为问题

| 行为 | 可能原因 | 应对策略 |
|------|---------|---------|
| **游走** | 焦虑、无聊、寻找某物/某人 | 安全环境、陪伴活动、转移注意力 |
| **攻击** | 恐惧、挫折、疼痛、误解 | 保持冷静、识别触发因素、转移注意力 |
| **拒绝照护** | 恐惧、不理解、不舒服 | 解释、分散注意力、选择时机 |
| **重复行为** | 焦虑、无聊、寻求安全感 | 分散注意力、提供活动、确保安全 |
| **夜间躁动** | 昼夜颠倒、疼痛、环境 | 白天活动、夜间安静、检查不适 |
| **藏东西** | 记忆障碍、安全感 | 不要指责、准备替代品、检查安全 |

#### 6.2 行为管理原则

**寻找原因**：
- 生理原因（疼痛、感染、便秘）
- 环境原因（噪音、光线、温度）
- 心理原因（焦虑、恐惧、无聊）

**非药物干预优先**：
- 环境调整
- 活动安排
- 沟通技巧
- 照护方式调整

**必要时就医**：
- 行为严重影响生活
- 可能伤害自己或他人
- 药物副作用

#### 6.3 应对激越和攻击

**预防**：
- 识别触发因素
- 保持环境稳定
- 建立规律

**应对**：
- 保持冷静
- 不要对抗
- 给予空间
- 转移注意力
- 确保安全

**事后**：
- 分析原因
- 调整照护方式
- 必要时就医

### 模块十一：紧急情况判断

#### 11.1 立即就医（拨打120）

- 🚨 高热不退（>39℃）
- 🚨 严重呼吸困难
- 🚨 意识障碍突然加重
- 🚨 严重外伤、出血
- 🚨 癫痫发作
- 🚨 误吸、窒息

#### 11.2 尽快就医（24小时内）

- ⚠️ 发热（>38℃）
- ⚠️ 咳嗽、咳痰（可能肺炎）
- ⚠️ 排尿困难、尿痛
- ⚠️ 便秘严重、腹痛
- ⚠️ 行为突然改变
- ⚠️ 跌倒、外伤
- ⚠️ 拒绝进食、饮水

#### 11.3 可观察（记录并复诊告知）

- 轻微行为改变
- 偶发遗忘加重
- 轻度食欲下降

### 模块十二：知识科普（去焦虑）

#### 12.1 阿尔茨海默症不是"老糊涂"

- 是疾病，不是正常老化
- 需要医学诊断
- 可以延缓进展
- 患者不是"故意的"

#### 12.2 照护可以改善生活质量

- 患者可以感受爱和关怀
- 熟悉的音乐、气味可以唤起记忆
- 简单的活动可以带来快乐
- 尊严照护很重要

#### 12.3 照护者不是"超人"

- 照护是长期工作，需要休息
- 寻求帮助是正常的
- 自己的情绪也很重要
- 有很多资源可以利用

#### 12.4 积极案例

> 许多阿尔茨海默症患者在科学照护下：
- 维持了多年的生活质量
- 与家人建立了新的连接方式
- 在熟悉的环境中感到安全
- 感受到了爱和尊严
