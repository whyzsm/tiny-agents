## 触发的能力

| 能力模块 | 能力 |
|---|---|
| `finance-control` | 收入、成本、现金流、预算和利润分析 |
| `revenue-growth` | 销售、渠道、定价、复购和增长动作 |
| `customer-compliance` | 客户合同、交付、隐私、合规和风险 |
| `operations-management` | 流程、人员、工具、供应商和执行节奏 |
| `decision-dashboard` | 经营仪表盘、周会/月会、指标和预警 |
| `action-tracking` | 行动项、负责人、期限、复盘和改进闭环 |

## 与已有专家团的边界

| 已有专家团 | 主要关系 |
|---|---|
| `finance-business-analysis` | 偏经营数据分析；本团覆盖经营管理动作和多角色协作。 |
| `consulting-one-person-company` | 偏一人公司；本团适合小企业综合管理。 |

## 我可以帮你做这些

1. 经营体检

   扫描财务、营收、客户、合规和运营风险。

2. 专家分诊

   把问题分配给财务、营收、客户合规和运营角色。

3. 决策建议

   输出优先级、资源取舍、负责人和时间表。

4. 指标看板

   建立经营指标、预警阈值和复盘节奏。

5. 闭环跟踪

   追踪行动项、结果、偏差和下一步。

## 完整交付物通常是

