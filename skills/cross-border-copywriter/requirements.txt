=============================================
跨境电商商品文案智能生成器（升级版）配置说明
=============================================

一、基础配置
1.  运行环境：Python 3.7+
2.  依赖库：无（原生Python库即可运行）
3.  编码格式：UTF-8
4.  运行方式：直接执行.py文件，或导入作为模块调用

二、输入参数配置（必填/可选说明）
| 参数名        | 类型    | 必填 | 说明                                                                 |
|---------------|---------|------|----------------------------------------------------------------------|
| product_name  | 字符串  | 是   | 商品名称（中文/英文均可，小语种生成时自动翻译适配）                   |
| category      | 字符串  | 否   | 品类：3C/服装/家居/美妆/户外/家电/饰品等                             |
| platform      | 字符串  | 否   | 目标平台：Amazon/Shopee/Temu/TikTok等（默认Amazon）                  |
| language      | 字符串  | 否   | 输出语言：en/zh/bilingual/es/fr/de/ja/ar（默认en）                   |
| keywords      | 数组    | 否   | 搜索关键词/SEO词列表（支持小语种关键词，自动融入文案）               |
| features      | 数组    | 否   | 商品卖点：材质、功能、尺寸、优势等（小语种生成时自动翻译卖点）       |
| tone          | 字符串  | 否   | 风格：professional/marketing/luxury/casual（默认professional）       |
| length        | 字符串  | 否   | 长度：short(标题)/medium(短描述)/long(详情)（默认long）              |
| ab_version    | 数字    | 否   | A/B测试版本数量：1-5（默认1个版本）                                  |
| ab_diff_type  | 字符串  | 否   | 版本差异类型：keyword/tone/structure/content（默认keyword）          |

三、平台字符限制配置（内置，可修改）
- Amazon：标题≤200字符，短描述≤150字符，详情页单段≤500字符，卖点单条≤100字符
- Temu/SHEIN：标题≤80字符，短描述≤100字符，详情页单段≤300字符
- TikTok Shop：标题≤60字符，短描述≤80字符，详情页单段≤200字符
- Shopee：标题≤120字符，短描述≤120字符，详情页单段≤400字符，标签≤20字符/个

四、合规词库配置（可扩展）
1.  侵权词：默认包含Nike、Apple等知名品牌词，可在代码compliance_words中添加/删除
2.  违禁词：按语种分类，包含夸大词、违规词，可根据平台规则更新
3.  小语种合规：适配各语种文化禁忌，如阿拉伯语禁止宗教敏感词，日语禁止夸大功效

五、输出配置
1.  单版本输出：包含标题、短描述、详细描述、卖点列表、标签、合规检查、字符适配信息
2.  多版本输出（A/B测试）：包含版本数量、差异类型、各版本完整文案信息
3.  输出格式：JSON格式，便于解析和调用

六、常见配置问题解决
1.  小语种翻译异常：检查language参数是否为指定值（es/fr/de/ja/ar）
2.  字符适配异常：检查platform参数是否正确，或修改platform_char_limit配置
3.  合规检查误判：更新compliance_words中的侵权词、违禁词列表
4.  A/B版本无差异：确认ab_version≥2，且ab_diff_type参数正确
