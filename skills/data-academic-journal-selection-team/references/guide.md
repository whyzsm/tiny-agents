## 触发的能力

| 能力模块 | 能力 |
|---|---|
| `manuscript-profiling` | 论文主题、学科、方法、创新点、质量和目标拆解 |
| `cn-journal-matching` | 中文期刊匹配、栏目范围、级别、审稿周期和命中率评估 |
| `sci-ssci-matching` | SCI/SSCI/Scopus 期刊匹配、分区、影响因子和范围校验 |
| `journal-risk-check` | 预警期刊、掠夺性期刊、停刊、黑名单和安全检测 |
| `submission-strategy` | 冲稳保分层、投稿顺序、改稿建议和时间计划 |
| `cover-letter-pack` | 投稿信、期刊对比表、风险说明和材料清单 |

## 与已有专家团的边界

| 已有专家团 | 主要关系 |
|---|---|
| `academic-literature-review` | 偏综述写作和文献整理；本团聚焦期刊匹配、投稿风险和投稿策略。 |
| `academic-deep-research-pro` | 偏深度学术研究；本团更像投稿前选刊顾问。 |

## 我可以帮你做这些

1. 稿件画像

   提取题目、摘要、关键词、学科、方法、创新点和投稿约束。

2. 双管道选刊

   中文刊和外文刊独立匹配，分别评估范围和风险。

3. 期刊情报

   整理分区、收录、影响因子、周期、版面费和近年主题。

4. 安全检测

   排查预警、掠夺、停刊、范围错配和投稿风险。

5. 策略输出

   给出冲稳保组合、投稿顺序、改稿建议和材料清单。
