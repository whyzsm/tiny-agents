# 自主优化架构师工作流

你现在要完成一项 `自主优化架构师` 任务。本包从 WorkBuddy `数据智能` 卡片转换而来，请按任务范围选择必要步骤，不要机械输出无关文档。

## 步骤 1：上下文收集

使用 `data-autonomous-optimization-architect-intake` 明确目标、输入、约束、对象、交付格式和判断标准。

## 步骤 2：策略设计

使用 `data-autonomous-optimization-architect-strategy` 基于 自动优化、系统调优、智能运维 设计路线、方案、结构和优先级。

## 步骤 3：执行生成

使用 `data-autonomous-optimization-architect-execution` 输出可直接使用的脚本、方案、清单、表格、内容或操作步骤。

## 步骤 4：质量检查

使用 `data-autonomous-optimization-architect-quality` 检查事实、合规、风险、边界、遗漏和可验证性。

## 步骤 5：指标复盘

使用 `data-autonomous-optimization-architect-measurement` 定义指标、验证证据、效果评估和复盘方法。

## 步骤 6：交付收口

使用 `data-autonomous-optimization-architect-handoff` 汇总最终产物、下一步动作、开放问题和沉淀模板。

## 最终输出

将以上步骤整合为 `自主优化架构师` 交付包，通常包含：

1. **任务背景和目标**
2. **关键假设和输入材料**
3. **策略/方案/流程**
4. **执行清单或可直接使用的内容**
5. **质量、风险和证据检查**
6. **下一步动作和复盘指标**
