## 触发的能力

| 能力模块 | 能力 |
|---|---|
| `excel-ingestion` | Excel/CSV 数据读取、字段理解、清洗和本地处理边界 |
| `trend-analysis` | 趋势、周期、增长、拐点和时间序列解释 |
| `structure-analysis` | 结构、占比、分层、分组和贡献度分析 |
| `anomaly-analysis` | 异常、离群、波动、缺失和可疑数据定位 |
| `parallel-synthesis` | 趋势/结构/异常三线并行汇总和一致性校验 |
| `deliverable-pack` | 网页、Excel、PPT、摘要和复盘建议交付 |

## 我可以帮你做这些

1. 文件接入

   识别 Excel/CSV 表结构、字段含义、数据质量和隐私边界。

2. 三专家并行

   分别做趋势、结构、异常分析并保留证据。

3. 综合判断

   交叉验证结论，区分确定发现、可能原因和待确认问题。

4. 成果打包

   按网页、Excel、PPT 或文字报告输出可交付内容。

5. 经营复盘

   把数据洞察转成一人公司可执行动作。

## 完整交付物通常是

```text
数据接入摘要
趋势分析
结构分析

综合结论
网页/Excel/PPT 交付说明
```
