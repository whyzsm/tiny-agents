# 品牌视觉专家团

`$design-brand-visual-team` 品牌视觉专家

## 触发的技能

| 技能 | 能力 |
|---|---|
| `$design-brand-visual-team` | 完整品牌视觉工作流总入口 |
| `$brand-cog` | 品牌定位、核心价值、品牌个性、语调、视觉识别框架 |
| `$visual` | 视觉调性、排版系统、图像风格、UI 应用建议 |
| `$theme-factory` | 品牌主色、辅色、点缀色、明暗模式、WCAG 对比度 |
| `$logo-creator` | 几何图元、负空间、极简 Logo、标准版和反白版 |
| `$poster` | 品牌海报、活动宣传图、社交媒体视觉模板 |
| `$svg-draw` | SVG 图标、插图、辅助图形、PNG 转换资源 |

## 我可以帮你做这些

1. 品牌策略和定位

   调用 `brand-cog`

   从品牌 Brief 梳理品牌定位、目标用户、核心价值、品牌个性和语调。

2. 视觉识别方向

   调用 `visual`

   把品牌定位转成视觉调性、排版层级、图像风格、插画方向和 UI 应用建议。

3. 品牌配色系统

   调用 `theme-factory`

   生成主色、辅色、点缀色、浅色/深色模式，并检查关键文本和背景的 WCAG 对比度。

4. Logo 设计方案

   调用 `logo-creator`

   设计极简 Logo 概念，输出标准版、反白版、单色版、安全区域和最小尺寸建议。

5. 品牌视觉物料

   调用 `poster`

   制作品牌主视觉海报、活动宣传图、社交媒体封面、头像模板和 HTML 视觉稿。

6. SVG 图形资源

   调用 `svg-draw`

   生成品牌图标、辅助图形、几何纹理、矢量插图和 PNG 导出资源。
