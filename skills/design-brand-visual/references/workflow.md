# 品牌视觉工作流

你现在要完成一项品牌视觉识别系统的设计任务。你已安装以下 Skill，请按步骤串联使用：

## 步骤 1：品牌圣经构建（获取层）
使用 **brand-cog** 完成：
- 基于品牌简报（Brief）梳理品牌定位和核心价值主张
- 定义品牌个性特质和语调风格（Tone of Voice）
- 构建视觉识别系统框架（色彩方向、字体方向、图像风格）
- 制定品牌规范文档（Brand Guideline）大纲
- 输出 Logo 设计方向和概念关键词

输出品牌圣经文档和视觉识别系统框架。

## 步骤 2：视觉设计指导与调性升级（分析层）
使用 **Visual** 完成：
- 基于品牌定位提供平面设计方向指引
- 评估和优化品牌视觉调性（高端/年轻/科技/温暖等）
- 制定排版系统规范（标题/正文/辅助文字层级）
- 定义图像和插画风格指南
- 提供 UI 交互场景下的品牌视觉应用建议

输出视觉设计指引文档。

## 步骤 3：品牌配色系统设计（分析层）
使用 **theme-factory** 完成：
- 基于品牌调性生成主色、辅色和点缀色方案
- 运用色彩和谐理论（互补色/类似色/三色）确保配色协调
- 检查配色的 WCAG 对比度合规性（文字/背景可读性）
- 输出多种格式的色值（HEX/RGB/HSL）
- 生成浅色/深色模式的配色变体

输出品牌配色方案和对比度检查报告。

## 步骤 4：Logo 设计（输出层）
使用 **Logo Creator** 完成：
- 运用几何图元和负空间技法设计品牌 Logo
- 确保 Logo 在不同尺寸下的可识别性（favicon 到海报级）
- 设计 Logo 的标准版、反白版和单色版
- 定义 Logo 安全区域和最小使用尺寸
- 生成矢量风格的极简 Logo 方案

输出 Logo 设计方案（含变体）。

## 步骤 5：品牌海报与视觉物料（输出层）
使用 **海报设计skill** 完成：
- 基于品牌视觉规范构建设计哲学（Algorithmic Philosophy）
- 设计品牌主视觉海报（品牌发布/活动/宣传）
- 制作社交媒体视觉物料（封面图/头像/配图模板）
- 应用品牌排版系统和配色方案
- 生成可渲染的 HTML 视觉稿

输出品牌海报和社交媒体视觉物料。

## 步骤 6：SVG 品牌图形资源（输出层）
使用 **SVG Draw** 完成：
- 生成品牌专属的矢量插图和图标
- 创建品牌辅助图形元素（装饰线条/几何图案/纹理）
- 制作可缩放的品牌图标集（SVG 格式）
- 将 SVG 转换为 PNG 格式供不同场景使用
- 确保图形风格与品牌视觉系统统一

输出 SVG 品牌图形资源包。

## 最终输出
将以上步骤的结果整合为完整的品牌视觉设计包，交付以下文件：
1. **品牌圣经**：品牌定位、个性、语调、视觉识别框架
2. **视觉设计指引**：排版系统、图像风格、调性规范
3. **品牌配色方案**：主色/辅色/点缀色、对比度报告、多模式变体
4. **Logo 设计方案**：标准版/反白版/单色版 + 使用规范
5. **品牌视觉物料**：海报、社交媒体模板
6. **SVG 图形资源**：品牌图标、辅助图形、矢量插图
