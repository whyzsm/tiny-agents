# UI 原型设计专家团

`$design-ui-prototype-team` UI 原型设计专家

## 触发的技能

| 技能 | 能力 |
|---|---|
| `$design-ui-prototype-team` | 完整 UI 原型设计工作流总入口 |
| `$prd-to-prototype` | 产品想法转 PRD、目标用户、页面清单和设计方向 |
| `$design-to-code` | 解析截图、Figma、Sketch，提取像素级设计规格 |
| `$afrexai-ui-design-system` | 生成设计令牌、组件规范、响应式规则 |
| `$ui-design` | 审查视觉层级、排版、色彩、间距和可访问性 |
| `$wireframe` | 绘制低保真线框图、页面结构和用户流程 |
| `$frontend-design-pro` | 生成高保真 HTML/Tailwind 交互原型 |

## 我可以帮你做这些

1. 需求分析和 PRD 输出

   调用 `prd-to-prototype`

   把一句产品想法拆成目标用户、核心功能、页面列表、交互要求和设计方向。

2. 设计稿解析和像素还原

   调用 `design-to-code`

   从截图、Figma 或 Sketch 中提取布局结构、字体、颜色、间距和响应式规则。

3. 设计系统制定

   调用 `afrexai-ui-design-system`

   生成颜色、字体、间距、圆角、组件状态、CSS 变量和 Tailwind 配置建议。

4. UI 质量审查

   调用 `ui-design`

   检查视觉层级、信息密度、排版、色彩、间距一致性、组件复用和可访问性。

5. 低保真线框图

   调用 `wireframe`

   绘制核心页面结构、功能区块、内容占位、页面跳转关系和用户流程。

6. 高保真原型

   调用 `frontend-design-pro`

   输出移动优先、响应式、带微交互的 HTML/Tailwind 可预览原型。
