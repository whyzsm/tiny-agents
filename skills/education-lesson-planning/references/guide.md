# 教案设计专家团

`$education-lesson-planning` 教案设计专家团总入口

## 触发的技能

| 技能 | 能力 |
|---|---|
| `$education-lesson-planning` | 教案设计专家团总入口 |
| `$teaching-plan-writer` | 专业教案写作助手，基于教学评一致性原则生成符合《义务教育课程标准（2022年版）》的个性化教案。 |
| `$teacher-assistant` | 教师助手全集Skill。覆盖教师日常教学全流程：教案生成、智能出题、成绩分析、家校沟通、教学总结 |
| `$classroom-lesson-pack` | 根据课程目标生成教案、互动题、作业与分层教学建议。；use for education, les |
| `$creator-course-outline` | 为创作者课程生成模块结构、单元目标、作业与里程碑。；use for course, creato |
| `$teacher-prep` | 教师备课助手，用于小学语文教学备课。支持古诗、现代文、寓言、童话等各类课文。当用户提出备课需求时 |
| `$ppt-lecture-notes` | 每周一键更新PPT讲师备注。从教案中提取讲师活动，自动写入对应PPT幻灯片备注。 |

## 我可以帮你做这些

你现在要完成一项教案设计任务。你已安装以下 Skill，请按步骤串联使用：

1. 专业教案撰写（获取层）

   使用 **teaching-plan-writer** 完成：
   - 基于教学评一致性原则设计教案
   - 符合《义务教育课程标准（2022年版）》要求
   - 明确教学目标、重难点和教学过程
   - 设计导入环节、新授环节、练习环节和总结环节
   - 生成个性化的完整教案文档

   输出符合新课标的完整教案初稿。

2. 教学工具与评估设计（获取层）

   使用 **新知绘宇教师助手** 完成：
   - 设计配套的评分标准（Rubric）
   - 规划课堂互动活动和教学策略
   - 制定学生评估方案和反馈机制
   - 准备家长沟通材料
   - 完善教案的评价维度

   输出评分标准、课堂活动方案和评估设计。

3. 分层教学与互动题设计（分析层）

   使用 **Classroom Lesson Pack** 完成：
   - 根据课程目标生成配套互动题
   - 设计分层教学建议适应不同学生水平
   - 制定课堂作业和课后练习
   - 确保教学内容与目标对齐
   - 生成完整的课堂教学资源包

   输出互动题库、分层教学方案和作业设计。

4. 课程模块与单元规划（分析层）

   使用 **Creator Course Outline** 完成：
   - 设计课程模块结构和教学单元
   - 明确每个单元的学习目标和能力要求
   - 规划作业节点和学习里程碑
   - 建立单元间的知识衔接关系
   - 生成结构化的课程大纲

   输出课程模块结构和单元规划文档。

5. 备课资料自动整理（输出层）

   使用 **teacher-prep** 完成：
   - 自动搜索课文相关资料（原文、作者、背景）
   - 整理生字词、重点段落和教学要点
   - 梳理课文知识结构和教学脉络
   - 补充拓展资源和参考材料
   - 输出结构化的备课资料包

   输出备课资料包和教学参考材料。

6. 教案PPT转化（输出层）

   使用 **教案内容写入PPT备注** 完成：
   - 从教案中提取各环节的讲师活动
   - 自动写入对应PPT幻灯片的备注区域
   - 确保教案内容与PPT页面一一对应
   - 支持一键批量更新备注内容
   - 输出带完整讲师备注的教学PPT

   输出带讲师备注的教学PPT文件。

## 完整交付物通常是

```text
将以上步骤的结果整合为完整的教案设计成果包，交付以下文件：
1. **完整教案**：教学评一致性、新课标、个性化教案
2. **评估工具**：Rubric评分标准、活动方案、评估设计
3. **分层资源包**：互动题、分层教学、作业设计
4. **课程大纲**：模块结构、单元目标、里程碑规划
5. **备课资料**：课文资料、生字词、教学参考
6. **教学PPT**：讲师备注、幻灯片对应、一键更新
```
