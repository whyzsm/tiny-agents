# 题库生成专家团

`$education-quiz-generation` 题库生成专家团总入口

## 触发的技能

| 技能 | 能力 |
|---|---|
| `$education-quiz-generation` | 题库生成专家团总入口 |
| `$quiz-creator` | Generate quizzes and assessments from any topic  |
| `$exam-generator` | 中国中小学试卷生成器。Use when teacher needs to create exam |
| `$math-edu-assistant` | 数学教材同步教育助手。本技能应在用户提出数学学习相关请求时触发，涵盖知识点讲解、 智能出题、作业 |
| `$exam-analyzer` | SKILL.md - 考点作战地图生成器 skill. Exam Analyzer skill  |
| `$k12-smart-teacher` | K12智能老师辅导系统，支持作业批改、错题分析、举一反三练习生成。覆盖小学/初中/高中全学段，语 |
| `$exam` | Generate practice tests, flashcards, study sched |

## 我可以帮你做这些

你现在要完成一项题库生成任务。你已安装以下 Skill，请按步骤串联使用：

## 步骤 1：多题型批量生成（获取层）
使用 **Quiz Generator** 完成：
- 根据知识点批量生成选择题、填空题和简答题
- 设定题目难度分级（基础/中等/拔高）
- 为每道题配备标准答案和详细解析
- 支持模拟考试模式的题目组织
- 建立题库并进行分类管理

输出分难度分题型的题库初版。

## 步骤 2：学科试卷组卷（获取层）
使用 **Exam Generator** 完成：
- 按照中国中小学考试标准生成完整试卷
- 确保题目覆盖教学大纲核心考点
- 控制试卷难度梯度和分值分布
- 规范试卷排版格式（题号/分值/答题区）
- 生成配套的评分标准和参考答案

输出规范化的学科试卷和评分标准。

## 步骤 3：教材同步智能出题（分析层）
使用 **数学教育助手** 完成：
- 严格按照教材内容同步出题
- 围绕每个知识点设计配套练习
- 对题目进行解题思路讲解
- 确保题目难度与教学进度匹配
- 支持小学至高中全阶段教材

输出教材同步练习题和解题讲解。

## 步骤 4：考点结构化与题型匹配（分析层）
使用 **考点分析专家** 完成：
- 从教学大纲或考试说明中提取核心考点
- 对每个考点标注重要等级（高频/中频/低频）
- 匹配每个考点适用的题型
- 生成考点作战地图
- 确保题库覆盖所有必考知识点

输出考点作战地图和题型匹配方案。

## 步骤 5：举一反三练习生成（输出层）
使用 **K12智能老师** 完成：
- 基于错题和薄弱点生成举一反三练习
- 覆盖小学/初中/高中全学段九大学科
- 自动批改练习并给出反馈
- 分析错误原因和知识盲区
- 生成个性化的强化练习题组

输出举一反三练习题和错题分析报告。

## 步骤 6：综合测试与评估生成（输出层）
使用 **Exam** 完成：
- 从学习材料中自动生成综合练习测试
- 制作知识点闪卡便于快速复习
- 生成配套学习计划和复习安排
- 支持计时模拟考试模式
- 输出完整的测试评估方案

输出综合测试卷、闪卡和学习计划。

## 最终输出
将以上步骤的结果整合为完整的题库生成成果包，交付以下文件：
1. **分级题库**：多题型、难度分级、答案解析
2. **规范试卷**：学科组卷、分值分布、评分标准
3. **同步练习**：教材同步、知识点配套、解题讲解
4. **考点地图**：考点提取、重要等级、题型匹配
5. **强化练习**：举一反三、错题分析、个性化练习
6. **综合测试**：练习测试、闪卡、学习计划、计时模拟
