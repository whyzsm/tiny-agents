## 我可以帮你做这些

你现在要完成一项学生评估任务。你已安装以下 Skill，请按步骤串联使用：

1. 评估方案与Rubric设计（获取层）

   使用 **新知绘宇教师助手** 完成：
   - 设计针对教学目标的评估方案
   - 制定多维度评分标准（Rubric）
   - 规划形成性评估和终结性评估策略
   - 设计学生反馈收集机制
   - 准备评估相关的家长沟通材料

   输出评估方案和评分标准文档。

2. 评分差距分析与提分计划（获取层）

   使用 **rubric-gap-analyzer** 完成：
   - 读取评分标准和作业要求
   - 对照Rubric分析学生作品的差距
   - 识别各评分维度的具体不足
   - 制定针对性的提分改进计划
   - 给出优先级排序的改进建议

   输出差距分析报告和提分计划。

3. 错题归类与薄弱环节诊断（分析层）

   使用 **Error Analysis** 完成：
   - 对学生错题进行系统归类
   - 定位每道错题对应的知识点
   - 分析薄弱环节和知识盲区
   - 绘制知识点掌握度图谱
   - 给出针对薄弱环节的复习建议

   输出错题分析报告和知识点图谱。

4. 全学段作业批改与反馈（分析层）

   使用 **K12智能老师** 完成：
   - 对作业进行智能批改和打分
   - 分析错误原因并归类错误类型
   - 生成举一反三的巩固练习
   - 覆盖小学/初中/高中九大学科
   - 输出详细的批改反馈和改进方向

   输出作业批改结果和个性化反馈。

5. 学员成长档案管理（输出层）

   使用 **student-growth-ops** 完成：
   - 建立学员个人学习档案
   - 记录每次学情评估结果和变化趋势
   - 整理家长反馈和沟通记录
   - 跟踪学习目标达成进度
   - 留档咨询记录和关键事件

   输出学员成长档案和学情追踪记录。

6. 成绩单评语生成（输出层）

   使用 **Xueersi Parent Comment Generator** 完成：
   - 根据学生表现生成温暖得体的评语
   - 适配成绩单、作业本和校园沟通等场景
   - 突出学生优点并委婉指出改进方向
   - 确保评语个性化、避免千篇一律
   - 输出可直接签字使用的家长评语

   输出个性化学生评语集。

## 完整交付物通常是

```text
将以上步骤的结果整合为完整的学生评估成果包，交付以下文件：
1. **评估方案**：Rubric评分标准、评估策略、反馈机制
2. **差距分析**：维度对标、不足识别、提分计划
3. **错题诊断**：错题归类、知识点图谱、复习建议
4. **批改反馈**：作业批改、错因分析、巩固练习
5. **成长档案**：学情记录、趋势追踪、沟通留档
6. **评语集**：成绩单评语、作业评语、家长沟通
```
