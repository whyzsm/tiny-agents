## 我可以帮你做这些

你现在要完成一项培训方案设计任务。你已安装以下 Skill，请按步骤串联使用：

1. 培训计划整体设计（获取层）

   使用 **IDA: Instructional Design Agent** 完成：
   - 明确培训目标和受众分析
   - 设计完整的课程体系和模块结构
   - 制定培训日程安排和时间规划
   - 建立培训效果评估机制
   - 准备证书模板和结业标准

   输出培训计划总方案和日程表。

2. 结构化员工培训计划（获取层）

   使用 **Employee Training Program** 完成：
   - 设计入职培训、合规培训和技能提升模块
   - 为每个模块制定学习目标和评估标准
   - 建立培训进度跟踪和完成度监测
   - 规划领导力发展和职业成长路径
   - 输出结构化的培训项目文档

   输出员工培训项目方案和跟踪表。

3. 专业培训课件开发（分析层）

   使用 **training-course-designer** 完成：
   - 针对企业培训需求设计专业课程
   - 一键生成包含课件在内的全套培训资料
   - 制作配套的营销文案和宣传材料
   - 确保课程内容与培训目标对齐
   - 输出可直接使用的培训课件包

   输出培训课件和配套资料。

4. 培训课程大纲与教学设计（分析层）

   使用 **教学设计** 完成：
   - 设计培训课程的详细大纲
   - 规划教学环节和互动方式
   - 制定效果评估方案和考核标准
   - 生成内部分享和交流材料
   - 确保教学策略与学习目标匹配

   输出课程大纲和教学设计文档。

5. 工作坊议程与日程规划（输出层）

   使用 **jackyshen-design-workshop-outline** 完成：
   - 生成结构化的工作坊大纲
   - 设计培训议程和时间分配
   - 规划每个环节的活动形式和产出物
   - 制定工作坊日程和分段脚本
   - 输出可直接执行的培训议程表

   输出工作坊大纲和培训议程。

6. 培训记录转化学习材料（输出层）

   使用 **Transcript to Content** 完成：
   - 将培训会议记录转化为结构化学习材料
   - 提取关键知识点生成学习指南
   - 制作速查表和检查清单
   - 整理可操作的复习内容
   - 输出完整的培训知识沉淀文档

   输出学习指南、速查表和知识沉淀文档。

## 完整交付物通常是

```text
将以上步骤的结果整合为完整的培训方案成果包，交付以下文件：
1. **培训计划**：目标分析、课程体系、日程安排、证书模板
2. **培训项目**：模块设计、评估标准、进度跟踪、发展路径
3. **培训课件**：专业课程、全套资料、营销文案
4. **教学设计**：课程大纲、教学环节、效果评估
5. **培训议程**：工作坊大纲、时间分配、分段脚本
6. **知识沉淀**：学习指南、速查表、检查清单、复习材料
```
