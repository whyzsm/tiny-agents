## 我可以帮你做这些

你现在要完成一项企业经营状况的深度分析任务。你已安装以下 Skill，请按步骤串联使用：

1. 获取经营数据（获取层）

   使用 **A股业绩快报查询** 完成：
   - 查询目标上市公司的业绩快报数据（支持沪深京）
   - 获取核心经营指标：营收、净利润、EPS、ROE
   - 拉取同比增长率，对比一季报、半年报、三季报、年报
   - 收集历史财务指标用于趋势分析

   将经营数据整理为结构化格式待用。

2. 自动化数据分析（分析层）

   使用 **Auto Data Analysis Claw** 完成：
   - 对财务报表（利润表、资产负债表、现金流量表）进行自动化分析
   - 执行 KPI 计算：毛利率、净利率、资产周转率、应收账款周转天数等
   - 进行同比环比分析，识别营收和利润的增长趋势
   - 执行多维度差异分析，定位业绩波动的核心驱动因素
   - 数据清洗和异常值标记

   输出数据分析底稿和 KPI 计算结果。

3. 杜邦体系深度拆解（分析层）

   使用 **financial roe analysis** 完成：
   - 基于杜邦分析体系对公司进行深度财务分析
   - 拆解 ROE 驱动因素：净利率 × 资产周转率 × 权益乘数
   - 评估盈利质量（扣非净利润占比、经营性现金流/净利润）
   - 分析资产运营效率和杠杆风险水平
   - 与行业均值和竞争对手对标

   输出杜邦分析报告和财务健康度评估。

4. 预算执行差异分析（分析层）

   使用 **Budget Vs Actual** 完成：
   - 对比月度/季度的预算与实际执行数据
   - 计算收入、费用和利润率的差异
   - 识别有利差异和不利差异
   - 分析差异产生的原因（量差、价差、结构差）
   - 标记需要管理层关注的重大偏差项

   输出预算执行差异分析报告。

5. KPI 模型与决策仪表盘（输出层）

   使用 **Business Intelligence** 完成：
   - 建立业务绩效模型，定义核心 KPI 体系
   - 将分析结果转化为面向管理层的决策仪表盘
   - 生成运营简报和经营分析周报/月报
   - 设定运营节奏（周例会/月度经营分析会的数据支撑）

6. 智能图表生成（输出层）

   使用 **smart-charts** 完成：
   - 读取分析数据（CSV/Excel/JSON），自动推荐最佳图表类型
   - 生成交互式 ECharts 图表（趋势图、对比图、占比图）
   - 制作营收趋势折线图、成本结构饼图、KPI 达成率仪表盘
   - 输出可嵌入报告的高质量可视化图表
