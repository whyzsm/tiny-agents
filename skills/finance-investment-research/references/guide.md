# 投研报告专家团

`$finance-investment-research` 投研报告专家团总入口

## 触发的技能

| 技能 | 能力 |
|---|---|
| `$finance-investment-research` | 投研报告专家团总入口 |
| `$ai-stock-analyst` | AI-powered Chinese A-share stock analyst. Fetche |
| `$investlog-ai` | 美股实时数据研究工具：股价行情、估值分析、财报数据、分析师评级、目标价、内部人交易、国会议员交易 |
| `$stock-research-engine` | 个股基本面深度研究引擎。当用户输入股票代码、公司名称或要求分析某只股票时触发。覆盖A股、港股、美 |
| `$industry-research-analyst` | 行业研究分析师 — 结构化深度行业研究skill。输入行业名称、产业链环节或研究课题，输出投行级 |
| `$valuation-analysis` | A股和港股价值投资分析系统。基于《股市真规则》方法论，提供完整的投资分析框架：护城河分析、财务健 |
| `$finance-research-report` | A股每周金融投研报告生成器 — 基于 AKShare / 新浪财经 / 东方财富数据源，自动生成 |

## 我可以帮你做这些

你现在要完成一份上市公司或行业的投研报告撰写任务。你已安装以下 Skill，请按步骤串联使用：

## 步骤 1：获取A股市场数据（获取层）
使用 **AI Stock Analyst** 完成：
- 通过 AkShare 获取目标A股公司的实时行情、技术指标和估值数据
- 拉取公司近期新闻和市场动态
- 获取评分投资建议作为初步参考

将A股市场数据保存待用。

## 步骤 2：获取美股/全球市场数据（获取层）
使用 **InvestLog AI** 完成：
- 获取美股目标公司的股价行情、财报数据和估值分析
- 拉取分析师评级、目标价和机构持仓数据
- 获取内部人交易、ETF持仓和技术指标信息
- 收集财务健康评分和分红拆股记录

将美股数据与步骤1的数据合并，形成完整的数据底稿。

## 步骤 3：个股基本面深度研究（分析层）
使用 **stock-research-engine** 完成：
- 以买方基金经理视角，对目标公司进行深度投资分析
- 输出包含市场情绪、基本面评估、管理层评估的分析简报
- 完成业务拆解、催化剂日历梳理和风险提示
- 覆盖A股、港股、美股的估值数据展示

记录个股分析的关键发现和投资逻辑。

## 步骤 4：行业深度研究（分析层）
使用 **Industry Research Analyst 行业研究分析师** 完成：
- 输出投行级别的行业深度研究报告
- 覆盖行业概览、市场规模和增长逻辑分析
- 梳理产业链上中下游关系和竞争格局
- 分析行业驱动因素、风险评估和投资建议

输出行业研究报告作为个股研究的宏观补充。

## 步骤 5：估值建模分析（分析层）
使用 **股票价值投资分析系统** 完成：
- 基于价值投资方法论进行完整估值分析
- 执行护城河分析（品牌、规模、网络效应、转换成本）
- 进行财务健康检查和会计质量评估
- 运用 DCF 模型计算内在价值，对比当前市价
- 整合管理层评估和行业分析，形成投资决策建议

输出估值分析报告和投资评级。

## 步骤 6：生成专业投研报告（输出层）
使用 **Skill Fin Report** 完成：
- 整合前五步的数据和分析结论
- 自动生成包含技术分析、交易信号、风险评估的专业 PDF 研报
- 报告包含行情走势图、关键财务指标图表
- 输出格式化的A股投研报告

## 最终输出
将以上步骤的结果整合为完整的投研报告包，交付以下文件：
1. **个股深度研究报告**：基本面分析、估值建模、投资评级
2. **行业研究报告**：行业概览、竞争格局、驱动因素
3. **专业投研报告PDF**：含技术分析图表、交易信号、风险评估的完整研报
