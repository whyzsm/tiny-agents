## 触发的能力

| 能力模块 | 能力 |
|---|---|
| `game-concept` | 游戏定位、核心循环、目标玩家和差异化 |
| `game-design` | 系统、关卡、数值、经济和剧情设计 |
| `technical-development` | 客户端、服务端、工具链、性能和平台适配 |
| `art-audio-production` | 美术风格、资产清单、UI、音频和动效需求 |
| `qa-playtest` | 测试计划、试玩反馈、缺陷、平衡性和体验验证 |
| `live-ops-launch` | 上线计划、运营活动、数据指标、版本迭代和留存优化 |

## 与已有专家团的边界

| 已有专家团 | 主要关系 |
|---|---|
| `tech-software-development` | 覆盖通用软件开发；本团加入游戏策划、美术、音频和运营。 |
| `design-prototype-handoff` | 覆盖界面交付；本团覆盖完整游戏生产流程。 |

## 我可以帮你做这些

1. 概念立项

   定义目标玩家、核心循环、平台、竞品和商业模式。

2. 设计开发

   拆解系统、技术、美术、音频、UI 和数据需求。

3. 资产生产

   输出资产清单、风格规范、音频需求和交付节奏。

4. QA 试玩

   制定测试路径、缺陷管理、平衡性验证和用户反馈。

5. 上线运营

   规划发布、运营活动、指标、留存和版本迭代。
