## ⚠️ 编辑重构原则（核心）

**核心原则**：重构的目标是"写出一篇好文章"，而非保留原文长度。

| 场景 | 编辑策略 | 示例 |
|-----|---------|------|
| **冗余废话** | 删除，保留核心 | "我们紧紧围绕...认真贯彻落实..." → 删除 |
| **重复内容** | 合并，保留最完整表述 | 两处提到同一成果 → 合并为一段最完整的 |
| **骨架章节** | 扩写 | 仅有标题"（三）人才队伍建设"无正文 → 扩写 200-400 字 |
| **逻辑断层** | 补充过渡 | 章节间跳跃 → 补充过渡句 |
| **数据缺失** | 合理推断补充 | "市场占有率提升" → 补充"约 X 个百分点" |
| **结构失衡** | 重组平衡 | 将 10 页压缩为 3 页的章节展开 |

**数据必须保留**：所有原文数字（金额/百分比/排名）必须出现，不得删除。

**扩写示例（旧骨架章节扩写）**：

```
【原文仅有标题】
（三）加强安全管理

【扩写后】⚠️ 以下为扩写内容，与用户工作紧密相关，应使用黄色底色标注
（三）安全管理基础持续夯实，安全防线进一步筑牢

围绕"全覆盖、零容忍、重实效"的工作要求，全面压实安全生产主体责任，建立健全风险分级管控和隐患排查治理双重预防机制。组织开展安全生产专项检查 X 次，排查整改各类隐患 X 项，整改完成率 100%。开展安全培训 X 场，覆盖员工 X 人次，员工安全意识显著提升。全年未发生安全生产责任事故，安全管理形势总体平稳。

---⚠️ 扩写内容标注说明：
- 在 Word 中，扩写内容使用黄色底色（#FFFF00）标注
- 判断标准：与用户具体岗位职责/业务深度结合的内容 → 黄色底色
- 通用过渡句、衔接语 → 无需标注
```

### 场景二：骨架章节扩写（反面典型 → 正面示范）

**反面典型（原文）：**

```
（三）强化品牌建设
（无正文）
```

**问题诊断：**
- 标题过于笼统，未体现具体举措和成效
- 完全没有正文，无法支撑任何实质性内容

**正向扩写（符合 Skill 三要素约束）：**

```
（三）品牌影响力显著提升，市场认知度不断增强

【思路】依靠质量提升和渠道拓展两大抓手，围绕目标客户需求精准发力；
【动作】一是举办品牌发布会 X 场，邀请行业媒体和重点客户参会；二是投放线上精准广告，累计曝光量达 X 万次；三是启动"千企体验计划"，邀请 X 家潜在客户实地参观。
【效果】品牌搜索指数同比提升 X%，客户主动咨询量增长 X%，品牌美誉度排名升至行业第 X 位。

---⚠️ 扩写内容标注说明：
- Word 中该段应使用黄色底色（#FFFF00）标注
- 理由：原文完全无正文，属于骨架扩写；扩写内容与用户业务深度绑定，非通用衔接语
```

**对比说明：**

| 维度 | 反面典型 | 正面示范 |
|------|---------|---------|

### 场景三：多稿冲突处理

**输入：**

```
【财务部数据】
营业收入增长 10%，成本下降 3%。

【市场部数据】
营业收入增长 12%，成本下降 5%。

【技术部叙述】
收入增长大概 10% 左右，成本控制效果还不错。
```

**冲突点识别（Step 1 结构化分析阶段）：**

```
冲突类型①：营业收入增速
- 财务部口径：增长 10%
- 市场部口径：增长 12%
- 技术部叙述：增长"大概 10% 左右"（模糊）

→ 建议处理方式：标注冲突，由用户确认最终口径
→ 若无法确认，默认采用"就高原则"或"多方印证取最优"

冲突类型②：成本下降幅度
- 财务部：3%
- 市场部：5%
- 技术部：效果"还不错"（无数据）

→ 建议处理方式：同①
```

**重构指令中的处理方式（Step 2 输出）：**

```
【多稿冲突处理指令】
1. 营业收入增速：两版本数据冲突（10% vs 12%）。建议正文采用"10%"，备注"最终数据以财务部口径为准"；或将两数据均列出："营业收入增长 10%-12%"。
2. 成本下降幅度：两版本数据冲突（3% vs 5%）。处理方式同上。
3. 技术部"大概 10%"等模糊表述：改为精确数字，无数据则注明"数据待确认"。
4. 若用户未反馈确认，采用"就高原则"并在脚注中注明"部分数据为初步统计，最终以主管部门核定为准"。
```

**重构后正文（多稿合一版）：**

```
## 二、经营质量稳步提升，主要指标符合预期

全年实现营业收入 X 亿元，同比增长 10%-12%（注：部分数据为初步统计，最终以财务部核定为准）；成本费用同比下降约 3%-5%，两重数据差异系统计口径不同所致，已与相关部门核实。
```

**质量检查：**
- 冲突数据已体现（10%-12%）
- 差异原因已说明（口径不同）
- 模糊表述已消除（去掉"大概"）
- 脚注已添加（数据溯源）

```
【原文仅有标题】
（三）人才队伍建设

【扩写后】⚠️ 以下为扩写内容，与用户工作紧密相关，应使用黄色底色标注
