# 《党政机关公文格式》（GB/T 9704-2012）核心规范

## 一、纸张与页面

| 项目 | 规格 |
|-----|------|
| 纸张 | A4 纸（210mm × 297mm） |
| 纸张颜色 | 白色（克重 60g/m² - 80g/m²） |
| 上边距 | 37mm |
| 下边距 | 35mm |
| 左边距 | 28mm |
| 右边距 | 26mm |
| 版心尺寸 | 156mm × 225mm（宽 × 高） |

## 二、正文格式

| 项目 | 规范 |
|-----|------|
| 字体 | 3 号仿宋体 |
| 字号 | 16pt（3 号） |
| 行距 | 28 磅固定值 |
| 每页行数 | 约 22 行 |
| 每行字数 | 约 28 字 |
| 文字颜色 | 黑色 |
| 英文字体 | 与相邻汉字一致 |

## 三、标题层次

| 层级 | 序号格式 | 字体字号 | 是否加粗 | 序号后空一字 |
|-----|---------|---------|---------|------------|
| 一级 | 一、二、三、 | 3 号黑体 | 是 | 是 |
| 二级 | （一）（二）（三） | 3 号仿宋体 | 否 | 是 |
| 三级 | 1. 2. 3. | 3 号仿宋体 | 否 | 是 |
| 四级 | （1）（2）（3） | 3 号仿宋体 | 否 | 是 |

## 四、公文标题

| 项目 | 规范 |
|-----|------|
| 字体 | 红色 2 号小标宋体 |
| 排布 |居中，排列使用梯形或菱形 |
| 字数 | 一般不超过 50 字 |
| 换行 | 不 break 单独成行 |

**标题公式**：
- `[发文机关]` + `[事由]` + `[文种]`
- 示例：`XX 公司关于 2026 年第一季度经营工作情况的报告`

## 五、公文红线

| 类型 | 位置 | 格式 |
|-----|------|------|
| 红色分隔线 | 发文字号下方 | 红色反线，4 号半角宋体字"━━━━━━" |
| 红色分隔线 | 主体与版记之间 | 红色一字线，宽度同版心 |

## 六、公文各要素位置

```
┌─────────────────────────────────────────────┐
│  [红线 - 红色分隔线 4 号宋体反线]            │
├─────────────────────────────────────────────┤
│                                             │
│  ××发〔2026〕5 号          ← 发文字号(红色) │
│                                             │
│  ════════════════════════  ← 红色分隔线     │
│                                             │
│     关于 XXX 的报告        ← 公文标题(红色)  │
│                                             │
│  XX 办公室：            ← 主送机关          │
│                                             │
│  正文内容...              ← 3 号仿宋体      │
│  一、XXX                  ← 3 号黑体        │
│  （一）XXX                ← 3 号仿宋体      │
│  1. XXX                   ← 3 号仿宋体      │
│                                             │
│  附件：1. XXX             ← 附件说明         │
│                                             │
│  ───────────────────────  ← 红色分隔线      │
│                                             │
│  抄送：XXX               ← 版记部分         │
│                                             │
│  XX 公司  XX 年 XX 月 XX 日印发             │
│                      ← 印发日期            │
└─────────────────────────────────────────────┘
```

## 七、其他要素

### 7.1 页码

| 项目 | 规范 |
|-----|------|
| 字体 | 4 号半角阿拉伯数字 |
| 位置 | 居中或右对齐 |
| 格式 | `-1-` 或 `第 1 页` |
| 封面不编页码 | 单独成页时，首页不显示页码 |

### 7.2 附件

```
附件：1. XXX
      2. XXX
```

- 附件标题与正文之间空一行
- 附件左上角加"附件"标识

### 7.3 签发人

```
签发人：XXX
```

- 位于发文字号同一行，右对齐
- 3 号仿宋体

### 7.4 引用顺序号

正文引用文件标题时，用"《》"标识；
引用顺序号（如"第一章""第二条"）时，用`「」`标识。

## 八、常用公文文种

| 文种 | 用途 |
|-----|------|
| **报告** | 向上级机关汇报工作、反映情况 |
| **请示** | 向上级机关请求指示、批准 |
| **通知** | 发布法规、转发公文、布置工作 |
| **函** | 机关之间商洽工作、询问答复 |
| **纪要** | 记载会议主要情况和议定事项 |

## 九、数字用法（GB/T 15835-2011）

| 类型 | 正确 | 错误 |
|-----|------|------|
| 分数 | 3/4 | 四分之三 |
| 小数 | 0.5 | 零点五 |
| 百分比 | 30% | 三成 |
| 倍数 | 增长 2 倍 | 增长两倍 |
| 概数 | 50 多人 | 50几个人 |
| 序列 | 第 1 名 | 第一名 |

## 十、常见错误规避

| 错误类型 | 正确做法 |
|---------|---------|
| 标题"关于..."开头 | 直接以事由开头，不加"关于" |
| 正文使用"我认为" | 使用"我们""本机关" |
| 口语化表达 | 使用规范的公文用语 |
| 层级序号混乱 | 严格按"一、（一）、1.、（1）" |
| 阿拉伯数字单独成行 | 数字与文字混排 |
| 页码使用全角数字 | 使用半角阿拉伯数字 |
