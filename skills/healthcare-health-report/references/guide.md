## 我可以帮你做这些

你现在要完成一项健康报告生成任务。你已安装以下 Skill，请按步骤串联使用：

1. 全维度健康数据采集（获取层）

   使用 **HealthFit-cn** 完成：
   - 收集运动训练、饮食营养和健康数据
   - 进行中医体质辨识和节气养生评估
   - 记录用户的运动计划和营养摄入
   - 整合中西医多维度健康信息
   - 建立全面的健康数据基线

   输出多维度健康数据采集报告。

2. 结构化健康摘要生成（获取层）

   使用 **Health Report** 完成：
   - 根据体征、症状和药物信息生成健康摘要
   - 整合生活方式数据（作息、运动、饮食）
   - 按标准格式组织健康信息
   - 生成结构化的健康状况概览
   - 标注关键健康指标和变化趋势

   输出结构化健康摘要报告。

3. 健康数据深度分析（分析层）

   使用 **Health Management** 完成：
   - 深度分析睡眠质量和睡眠模式
   - 评估运动表现和恢复状态
   - 识别健康数据中的异常趋势
   - 进行多维健康指标关联分析
   - 生成数据驱动的健康洞察

   输出健康数据分析报告和趋势洞察。

4. 体检报告通俗解读（分析层）

   使用 **Health Checkup Report** 完成：
   - 用通俗中文解读体检报告和化验结果
   - 标注异常指标及其可能含义
   - 评估异常项的紧急程度
   - 列出需要随访的问题清单
   - 给出后续检查和就医建议

   输出体检报告通俗解读文档。

5. 个人健康档案记录（输出层）

   使用 **Personal Health Journal** 完成：
   - 按日期生成结构化健康记录
   - 记录体温、血压、心率、血氧等体征
   - 跟踪排便、疼痛等身体变化
   - 输出每日健康总结
   - 进行风险分级和预警提示

   输出个人健康档案和每日总结。

6. 健康数据持续跟踪管理（输出层）

   使用 **Health Manager** 完成：
   - 持续跟踪血压、心率等核心指标
   - 管理用药记录和服药提醒
   - 分析症状变化趋势
   - 提供个性化健康建议
   - 输出长期健康管理方案

   输出健康跟踪管理方案和建议。

## 完整交付物通常是

```text
将以上步骤的结果整合为完整的健康报告成果包，交付以下文件：
1. **健康数据**：多维度采集、中西医融合、数据基线
2. **健康摘要**：结构化报告、指标概览、变化趋势
3. **数据分析**：睡眠/运动/恢复分析、异常趋势、关联洞察
4. **体检解读**：通俗解读、异常标注、紧急程度、随访问题
5. **健康档案**：结构化记录、每日总结、风险分级
6. **管理方案**：指标跟踪、用药管理、个性化建议
```
