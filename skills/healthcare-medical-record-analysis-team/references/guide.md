## 我可以帮你做这些

你现在要完成一项病历分析任务。你已安装以下 Skill，请按步骤串联使用：

1. 病历结构化整理（获取层）

   使用 **腾讯健康-AI临床助手-智能病例** 完成：
   - 将非结构化病历文本转换为规范化电子病历
   - 提取主诉、现病史、既往史等核心信息
   - 按标准模板组织病历各部分内容
   - 确保病历信息完整性和规范性
   - 生成结构化的电子病历文档

   输出规范化的结构化电子病历。

2. 病历文本规范化（获取层）

   使用 **unisound-medical-term-normalization** 完成：
   - 将口语化病历描述转换为规范医学记录
   - 统一医学术语、字段结构和临床表述
   - 补齐主诉、现病史、体格检查、辅助检查和诊疗计划等关键字段
   - 标注仍需医生确认或补录的信息
   - 输出符合临床书写规范的结构化记录

   输出规范化的病历记录。

3. 病历文本辅助诊断（分析层）

   使用 **腾讯健康-AI临床助手-辅助诊断** 完成：
   - 分析病历文本和实验室检查结果
   - 提供系统化的诊断推理过程
   - 列出鉴别诊断和可能性排序
   - 提供循证医学依据的决策支持
   - 标注需要重点关注的异常指标

   输出辅助诊断分析报告和推理过程。

4. 医疗实体识别与提取（分析层）

   使用 **Medical Entity Extractor** 完成：
   - 从患者信息中提取症状实体
   - 识别药物名称、剂量和用法
   - 提取化验值和检查结果数据
   - 标注诊断编码和疾病分类
   - 构建患者医疗实体关系图

   输出医疗实体提取结果和关系图。

5. 医疗文档通俗解读（输出层）

   使用 **Patiently AI** 完成：
   - 将专业医疗文档转化为通俗易懂的说明
   - 解释检查结果和化验报告的含义
   - 翻译处方和用药说明
   - 解读出院小结和随访建议
   - 输出患者友好的医疗文档解读

   输出通俗化的医疗文档解读报告。

6. 症状分析与就医建议（输出层）

   使用 **Medical Advice** 完成：
   - 根据症状和病史分析可能的原因
   - 提供是否需要就医的初步判断
   - 推荐适合的就诊科室
   - 建议可能需要的检查项目
   - 提供常见疾病的用药指导参考

   输出症状分析和就医建议报告。

## 完整交付物通常是

```text
将以上步骤的结果整合为完整的病历分析成果包，交付以下文件：
1. **结构化病历**：规范化电子病历、核心信息提取
2. **入院记录**：三步流程、体格检查、初步诊断
3. **辅助诊断**：诊断推理、鉴别诊断、异常标注
4. **实体提取**：症状/药物/化验值/诊断实体图
5. **文档解读**：通俗解释、检查报告、用药说明
6. **就医建议**：症状分析、科室推荐、检查建议
```
