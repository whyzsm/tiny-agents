## 我可以帮你做这些

你现在要完成一项护理方案设计任务。你已安装以下 Skill，请按步骤串联使用：

1. 护理理论选择与论证（获取层）

   使用 **nursing-theory-matcher（护理研究-理论支持论证工具）** 完成：
   - 根据护理研究题目匹配适用的护理理论
   - 提供理论依据和文献支撑
   - 帮助构建护理方案的理论框架
   - 进行护理科研的理论论证
   - 为护理计划奠定循证基础

   输出护理理论匹配报告和论证框架。

2. 临床护理计划制定（获取层）

   使用 **Nurse** 完成：
   - 评估患者状况制定个性化护理计划
   - 完善护理文档记录和交接班信息
   - 规划患者沟通策略和健康宣教
   - 制定护理操作流程和时间安排
   - 建立护理问题清单和干预措施

   输出个性化护理计划和文档模板。

3. 专病照护方案设计（分析层）

   使用 **阿尔茨海默症照护助手** 完成：
   - 提供疾病认知和发展阶段评估
   - 制定日常照护和安全管理方案
   - 设计有效的沟通技巧和行为管理策略
   - 规划照护者自身的支持和减压措施
   - 对接医疗资源和社会支持

   输出专病照护方案和照护者支持计划。

4. 护理监护路径规划（分析层）

   使用 **何药师护理监护路径规划** 完成：
   - 根据诊断、症状或病历信息制定护理监护路径
   - 检索护理操作规范并结构化为执行方案
   - 梳理每日巡查、风险因素、观察指标和处置建议
   - 标注需要人工确认的患者信息和风险阈值
   - 输出完整护理监护计划

   输出护理监护路径和风险观察清单。

5. 康复方案制定（输出层）

   使用 **Bio Reabilita Z** 完成：
   - 针对复杂损伤制定高级康复方案
   - 结合生物特征分析优化恢复路径
   - 规划神经营养支持和物理治疗方案
   - 设定康复阶段目标和评估节点
   - 输出个性化的康复训练计划

   输出高级康复方案和训练计划。

6. 护理信息通俗化转译（输出层）

   使用 **Lay Summary Gen** 完成：
   - 将复杂医学和护理信息转化为通俗语言
   - 确保患者和家属能够理解护理方案
   - 保持科学性的同时提升可读性
   - 制作患者友好的护理指导手册
   - 输出照护者可直接使用的参考文档

   输出通俗化护理指导手册。

## 完整交付物通常是

```text
将以上步骤的结果整合为完整的护理方案成果包，交付以下文件：
1. **理论框架**：护理理论匹配、文献支撑、循证基础
2. **护理计划**：个性化方案、文档模板、干预措施
3. **专病照护**：疾病评估、日常照护、照护者支持
4. **健康监测**：数据监测、异常预警、趋势跟踪
5. **康复方案**：高级康复、训练计划、阶段目标
6. **护理手册**：通俗转译、患者友好、照护者参考
```
