## 我可以帮你做这些

你现在要完成一项会议纪要的生成与整理任务。你已安装以下 Skill，请按步骤串联使用：

1. 会议纪要模板化生成（获取层）

   使用 **Meeting Minutes Pro** 完成：
   - 根据会议类型选择合适模板（站会/周会/复盘会/决策会）
   - 记录会议基本信息（时间/地点/参会人/主持人）
   - 按模板结构填充会议内容和讨论要点
   - 提取行动项（Action Items）和待办事项
   - 输出标准格式的会议纪要初稿

   输出模板化会议纪要初稿。

2. 笔记转行动项（获取层）

   使用 **AI Meeting Notes w/ Action Items + To-Do List Tracker** 完成：
   - 将凌乱的会议笔记或转录文本快速清洗整理
   - 提取带负责人和截止日期的行动项
   - 自动生成会议摘要
   - 保存并建立可搜索的会议记录库
   - 集成待办追踪，确保跟进到位

   输出清洗后的会议摘要和行动项清单。

3. 决策型会议深度整理（分析层）

   使用 **Meeting Note** 完成：
   - 梳理探讨/决策型会议的完整脉络
   - 提炼结论、共识、分歧和决策轨迹
   - 识别隐含假设、风险机会
   - 提取可执行行动项并标注优先级
   - 通过 Zettelkasten 方法建立知识连接

   输出决策型会议的深度纪要和知识图谱。

4. 原始记录结构化整理（分析层）

   使用 **会议纪要自动整理工具** 完成：
   - 将原始会议记录智能整理为结构化纪要
   - 按议题分类归纳讨论内容
   - 提取决议事项、责任人和截止日期
   - 标注关键决策和待确认事项
   - 输出格式规范的正式会议纪要

   输出结构化正式会议纪要。

5. 音频转写与文档输出（输出层）

   使用 **Meeting Notes Assistant** 完成：
   - 使用本地 Whisper 进行会议录音转写（离线、隐私安全）
   - 生成结构化纪要（时间/议题/结论/待办/关键词）
   - 提取 Action Items 并分配责任人
   - 支持 Word/PDF/邮件格式输出
   - 实现会议归档与待办分发

   输出转写文本和多格式会议纪要文档。

6. 多平台会议记录归档（输出层）

   使用 **DingTalk Minutes** 完成：
   - 自动整理钉钉/飞书等平台的会议记录
   - 智能合并相关会议内容
   - 提取参会人、核心议题、关键决策和待办事项
   - 分类归档到指定目录
   - 建立可检索的会议知识库

## 完整交付物通常是

```text
将以上步骤的结果整合为完整的会议纪要包，交付以下文件：
1. **模板化纪要初稿**：标准格式、基本信息、讨论要点
2. **行动项清单**：负责人、截止日期、待办追踪
3. **决策型深度纪要**：结论/共识/分歧、决策轨迹、知识连接
4. **结构化正式纪要**：议题分类、决议事项、格式规范
5. **多格式文档**：音频转写、Word/PDF 输出、归档分发
6. **多平台归档**：钉钉/飞书记录整理、分类归档、知识库
```
