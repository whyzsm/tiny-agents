# 简历筛选专家团

`$hr-resume-screening` 简历筛选专家团总入口

## 触发的技能

| 技能 | 能力 |
|---|---|
| `$hr-resume-screening` | 简历筛选专家团总入口 |
| `$resume-screening` | 专业的简历筛选与评估技能，按照"先硬后软，先筛后读"的漏斗式流程，从基础匹配度、能力匹配度、动机 |
| `$resume-parser` | 智能简历解析系统，支持PDF/Word/图片格式简历的结构化信息提取、岗位匹配度分析、优化建议生 |
| `$resume-screener-pro` | 完整招聘解决方案 - 四阶段一站式流程（简历筛选→面试设计→面试评估→最终推荐）。整合Topgr |
| `$easy-recruitment` | 智能招聘助手。根据职位JD自动筛选和评估简历，输出候选人排序报告，并为每个候选人生成定制化面试问 |
| `$applicant-screening-zh` | 简历筛选 / 招聘筛选 / 候选人评估 / resume screening / applica |
| `$interview-designer` | Analyze resumes and design interview strategies  |

## 我可以帮你做这些

你现在要完成一项简历筛选与候选人评估任务。你已安装以下 Skill，请按步骤串联使用：

## 步骤 1：简历量化筛选（获取层）
使用 **专业简历筛选技能** 完成：
- 按"先硬后软，先筛后读"的漏斗式流程筛选简历池
- 从基础匹配度、能力匹配度、动机稳定匹配度三个维度量化评估
- 对候选人进行初步分级（强匹配/一般匹配/不匹配）
- 标记简历中的关键亮点和风险信号
- 生成初筛通过名单和淘汰原因说明

输出初筛评估报告和候选人分级名单。

## 步骤 2：简历结构化解析（获取层）
使用 **resume-parser** 完成：
- 解析候选人简历文件（PDF/Word/图片格式）
- 提取核心结构化信息（教育背景/工作经历/技能/项目经验）
- 计算简历与目标岗位 JD 的匹配度得分
- 识别简历中的信息缺失和不一致之处
- 导出标准化候选人数据供后续环节使用

输出结构化简历数据和 JD 匹配度分析。

## 步骤 3：四阶段招聘评估（分析层）
使用 **Resume Screener Pro** 完成：
- 执行简历深度筛选（Topgrading 方法论）
- 设计结构化面试方案和评分标准
- 进行绩效预测和 ATS 优化评估
- 控制招聘偏见（bias control）
- 生成候选人综合评估和最终推荐排序

输出四阶段评估报告和候选人推荐排序。

## 步骤 4：JD 智能匹配与排序（分析层）
使用 **recruitment-assistant** 完成：
- 根据职位 JD 自动筛选和评估全部简历
- 输出候选人匹配度排序报告
- 为每位候选人生成定制化面试问题清单
- 标注候选人的核心优势和潜在风险
- 给出各候选人的录用优先级建议

输出候选人排序报告和定制面试问题。

## 步骤 5：批量简历评分排名（分析层）
使用 **Applicant Screening Zh** 完成：
- 批量解析大量简历（支持 1000+ 份）
- 按岗位要求进行无偏见自动评分
- 生成候选人排名和评分明细
- 提供多维度数据分析（技能分布/经验分布/学历分布）
- 输出可视化筛选报告

输出批量评分排名和数据分析报告。

## 步骤 6：循证面试策略设计（输出层）
使用 **Interview Designer** 完成：
- 基于简历分析设计针对性面试策略
- 制定结构化面试指南和评分标准
- 设计胜任力验证问题（行为面试/情境面试）
- 融合 Topgrading 方法论和绩效导向招聘理论
- 准备深度追问清单和评估维度

## 最终输出
将以上步骤的结果整合为完整的简历筛选包，交付以下文件：
1. **初筛评估报告**：候选人分级、匹配度评分、淘汰原因
2. **结构化简历数据**：核心信息提取、JD 匹配度分析
3. **四阶段评估报告**：Topgrading 评估、绩效预测、推荐排序
4. **候选人排序报告**：JD 匹配排序、定制面试问题、录用优先级
5. **批量评分报告**：评分明细、多维数据分析、可视化报告
6. **面试策略方案**：结构化面试指南、胜任力验证问题、追问清单
