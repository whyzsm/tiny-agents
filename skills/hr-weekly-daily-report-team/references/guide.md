## 我可以帮你做这些

你现在要完成一项周报日报的生成任务。你已安装以下 Skill，请按步骤串联使用：

1. 工作报告全类型生成（获取层）

   使用 **周报/日报生成与润色** 完成：
   - 根据报告类型选择模板（日报/周报/月报）
   - 收集本周期工作内容、成果和数据
   - 按标准结构填充报告各板块
   - 自动生成工作总结和下期计划
   - 支持多种报告风格切换

   输出标准格式的工作报告初稿。

2. 日报周报结构化生成（获取层）

   使用 **Chinese Daily Report Generator** 完成：
   - 根据工作记录自动生成结构化日报/周报
   - 按职场规范格式化输出
   - 包含问题复盘板块（遇到的问题/解决方案/经验教训）
   - 自动提炼工作亮点和关键成果
   - 输出可直接发送的日报/周报文本

   输出结构化日报/周报和问题复盘。

3. 飞书数据自动汇总（分析层）

   使用 **飞书周报** 完成：
   - 调用飞书 API 拉取指定时间范围的聊天记录
   - 读取本地 daily memory 日志
   - 合并两个数据源的工作内容
   - 按用户指定的周报模板自动整理
   - 输出飞书格式的周报

   输出飞书周报和数据汇总。

4. 智能提炼关键信息（分析层）

   使用 **Report Generator** 完成：
   - 根据职业角色定制报告风格和重点
   - 从本周工作内容中提炼关键信息
   - 生成具有洞察力的分析和建议
   - 包含问题分析与下周计划
   - 输出专业且结构化的周报

   输出智能提炼的专业周报。

5. 工作汇报 PPT 生成（输出层）

   使用 **工作报告PPT生成技能** 完成：
   - 使用 python-pptx 创建专业演示文稿
   - 按规范格式排版（标题宋体48号/内容36号/正文18号）
   - 控制总页数在 10 页以内
   - 支持工作周报、月报、项目汇报等类型
   - 输出可直接使用的 PPT 文件

   输出工作汇报 PPT 文件。

6. 多格式多平台输出（输出层）

   使用 **智能报告生成助手** 完成：
   - 自动分析数据并生成专业报告
   - 支持日报、周报、月报、分析报告等多种类型
   - 输出到飞书文档或本地文件
   - 适配不同平台的格式要求
   - 确保报告质量和格式一致性
