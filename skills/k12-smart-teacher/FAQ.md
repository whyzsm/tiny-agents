# 常见问题与排障

## 1. 生成文件相关

| 问题 | 可能原因 | 处理办法 |
|------|----------|----------|
| Word 生成失败 | 未安装 `python-docx` | 脚本会自动生成 Markdown；如必须要 Word，运行 `python3 scripts/quick_setup.py` 后重试 |
| 输出路径没有后缀 | 不知道生成什么格式 | 脚本会自动改成 `.md` |
| 输出目录不存在 | 例如 `练习/小明.md` 中 `练习` 文件夹还没有 | 脚本会自动创建目录 |
| JSON 看起来不像试卷 | JSON 是给系统处理的结构化数据 | 直接打印建议使用 `.md` 或 `.docx` |

## 2. 出题内容相关

| 问题 | 可能原因 | 处理办法 |
|------|----------|----------|
| 题目不够贴合教材 | 未提供教材版本、章节、原题材料 | 先生成通用练习；补充“地区+教材版本+章节/原题”后再细化 |
| 主题不在内置列表 | 均衡题库暂未覆盖该知识点 | 使用学科兜底模板；后续可把主题补到 `references/balanced_topic_bank.json` |
| 开放题没有唯一答案 | 语文、历史、政治等学科常见 | 按“观点明确、依据充分、表达完整”评分 |
| 数学/理科计算题担心答案 | 复杂题可能需要二次验算 | 先做公式、单位、数量级检查，再给学生使用 |

## 3. 作业批改相关

| 问题 | 可能原因 | 处理办法 |
|------|----------|----------|
| 图片看不清 | 拍摄模糊、反光、题目被遮挡 | 请用户补发清晰照片，或把题目文字打出来 |
| 无法判断年级 | 题目材料缺少年级特征 | 询问“孩子几年级、上/下学期、教材版本” |
| 题目跨学科 | 综合题同时涉及多个知识点 | 先判断主要考查点，再标注辅助学科 |
| 用户只想要答案 | 可能赶时间 | 可以先给结论，但必须附2-4步最短讲解和一道同类题 |

## 4. 视频推荐相关

| 问题 | 可能原因 | 处理办法 |
|------|----------|----------|
| 搜不到稳定视频链接 | 联网工具不可用或搜索结果不稳定 | 给可复制关键词，例如“浮力 初中 讲解 B站” |
| 视频打不开 | 平台限制或链接失效 | 换 B站、学而思、洋葱学园等国内平台关键词 |
| 视频太难 | 学段不匹配 | 搜索词加年级，例如“五年级 最大公因数 动画讲解” |

## 5. 推荐回复模板

### 图片不清

```text
这张图里有几处看不清，我担心直接批改会误判。
可以请你补发一张更清晰的照片，或者把第X题文字打出来吗？
我可以先根据目前能看清的部分讲解相关知识点。
```

### 主题不够具体

```text
我可以先生成一套通用练习。想更贴合教材的话，请补充：
1. 孩子几年级
2. 使用哪个教材版本
3. 具体章节或原题文字
```

### Word 失败

```text
Word 依赖暂不可用，我已自动生成可打印的 Markdown 版。
如果必须要 Word，可以先运行 python3 scripts/quick_setup.py，再重新生成。
```
