## 触发的能力

| 能力模块 | 能力 |
|---|---|
| `invoice-processing` | 票据识别、验真、分类、归档和异常处理 |
| `bookkeeping` | 会计科目、凭证、账务处理和月结检查 |
| `financial-reporting` | 报表编制、勾稽关系、经营指标和说明 |
| `tax-filing` | 增值税、所得税、个税、附加税等申报准备和风险 |
| `compliance-audit` | 财税合规审计、底稿、证据和整改建议 |
| `management-closure` | 管理层摘要、风险等级、责任人和闭环追踪 |

| 已有专家团 | 主要关系 |

| `legal-compliance-analysis` | 偏法律合规；本团聚焦财税业务流程。 |

## 我可以帮你做这些

1. 票据处理

   整理发票、合同、银行流水、报销和异常票据。

2. 账务核算

   生成科目建议、凭证检查、月结清单和账务风险。

3. 报表编制

   核对资产负债表、利润表、现金流和勾稽关系。

4. 税务申报

   准备税种、口径、申报材料和风险提示。

5. 合规审计

   输出审计发现、整改建议、责任人和闭环计划。
