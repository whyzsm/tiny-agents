# 广告文案专家团

`$marketing-ad-copywriting-team` 广告文案专家团总入口

## 触发的技能

| 技能 | 能力 |
|---|---|
| `$marketing-ad-copywriting-team` | 广告文案专家团工作流总入口 |
| `$marketing-skills` | 营销策略、目标受众、渠道选择、CRO/SEO/广告发布和分析框架 |
| `$copywriting` | 落地页、邮件、广告、销售页等英文或通用说服型文案生成 |
| `$bytesagain-copywriter-cn` | 中文和英文营销文案、AIDA、A/B 测试和中文标题公式 |
| `$ad-copywriter` | 信息流、搜索、电商和多平台广告文案生成与 A/B 变体 |
| `$landing-page-generator-openclaw` | 高转化、移动端适配的落地页内容和页面结构生成 |
| `$ecommerce-copywriter` | 淘宝、拼多多、抖音、京东商品标题、详情页、卖点和促销文案 |

## 我可以帮你做这些

1. 营销策略和受众分析

   调用 `marketing-skills`

   明确投放目标、目标受众、关键卖点、渠道策略、文案语调和转化路径。

2. 文案框架和说服力构建

   调用 `copywriting`

   运用 AIDA、PAS、FAB 等框架设计标题、正文、卖点、证明和 CTA。

3. 中文营销文案撰写

   调用 `bytesagain-copywriter-cn`

   按中文平台语境生成标题、开头钩子、正文论证、客户证言和 A/B 版本。

4. 全平台广告文案生成

   调用 `ad-copywriter`

   生成信息流、朋友圈、搜索、直通车、巨量引擎、Google Ads 等平台适配文案。

5. 高转化落地页生成

   调用 `landing-page-generator-openclaw`

   生成首屏、CTA、社会认同、功能利益点和移动端友好的落地页结构。

6. 电商爆款文案

   调用 `ecommerce-copywriter`

   生成商品标题、详情页、核心卖点、促销活动文案和平台适配建议。

7. 端到端广告文案交付

   调用 `marketing-skills`、`copywriting`、`bytesagain-copywriter-cn`、`ad-copywriter`、`landing-page-generator-openclaw`、`ecommerce-copywriter`

   从策略、受众、核心文案到平台适配、落地页和电商文案完整串联。

## 完整交付物通常是

```text
营销策略简报：投放目标、受众画像、卖点定位、渠道策略

核心文案库：标题变体、正文模板、卖点表达、CTA 变体

全平台广告文案：信息流、搜索、电商平台适配版本和 A/B 变体

高转化落地页：首屏文案、页面结构、CTA、社会认同和移动端建议

电商文案集：商品标题、详情页、促销文案和平台规则适配
```
