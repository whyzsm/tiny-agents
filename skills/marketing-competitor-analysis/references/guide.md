# 竞品分析专家团

`$marketing-competitor-analysis` 竞品分析专家团总入口

## 触发的技能

| 技能 | 能力 |
|---|---|
| `$marketing-competitor-analysis` | 竞品分析专家团工作流总入口 |
| `$market-research` | 市场规模估算、市场细分、竞品图谱、价格核查和需求验证 |
| `$market-analysis-cn` | 中文市场趋势、企业市场变化、用户行为洞察和行业风险分析 |
| `$competitive-intelligence-market-research` | B2B SaaS 场景竞争情报、产品功能、技术栈和商业模式分析 |
| `$competitor-analysis-report` | SWOT、功能矩阵、价格对比、定位分析和结构化竞品报告 |
| `$competitive-product-research` | 体验对标、数据对标、策略对标和 HTML 竞品分析报告 |
| `$competitor-watch` | 竞品网站、产品页、定价页、新功能和内容变更持续监控 |

## 我可以帮你做这些

1. 市场基础调研

   调用 `market-research`

   估算 TAM/SAM/SOM，梳理市场细分、主要玩家、价格区间和需求缺口。

2. 中文市场趋势洞察

   调用 `market-analysis-cn`

   分析目标企业或行业趋势、用户行为、增长机会、风险因素和中文市场特征。

3. B2B 竞争情报收集

   调用 `competitive-intelligence-market-research`

   收集竞品功能、技术栈、客户案例、商业模式、增长策略和竞争差距。

4. SWOT 和结构化对比

   调用 `competitor-analysis-report`

   生成 SWOT、功能矩阵、价格对比、定位差异和战略建议。

5. 竞品分析报告输出

   调用 `competitive-product-research`

   整合体验、数据和策略对标，生成可用于评审会或复盘会的 HTML 报告。

6. 竞品持续监控

   调用 `competitor-watch`

   配置竞品网站、产品页、定价页和发布动态监控，输出预警规则和动态摘要。

7. 端到端竞品分析交付

   调用 `market-research`、`market-analysis-cn`、`competitive-intelligence-market-research`、`competitor-analysis-report`、`competitive-product-research`、`competitor-watch`

   从市场调研、竞争情报、结构化对比到报告输出和持续监控完整串联。

## 完整交付物通常是

```text
市场规模与趋势报告：TAM/SAM/SOM、行业趋势、用户行为洞察

竞品图谱和竞争情报摘要：主要玩家、产品能力、技术栈、商业模式

SWOT 分析与对比矩阵：功能、价格、定位、目标客群和差异化机会

HTML 竞品分析报告：体验对标、数据对标、策略对标和可执行建议

竞品监控方案：监控对象、监控频率、预警规则和动态摘要
```
