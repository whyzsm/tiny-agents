# 活动策划专家团

`$marketing-event-planning` 活动策划专家团总入口

## 触发的技能

| 技能 | 能力 |
|---|---|
| `$marketing-event-planning` | 活动策划专家团工作流总入口 |
| `$afrexai-event-planner` | 会议、发布会、展览、研讨会和企业活动的整体策划与效果衡量 |
| `$campaign-planning` | 活动策划可落地指南、SOP、阶段清单和执行责任拆解 |
| `$afrexai-event-planning` | 活动定价、预算、供应商、后勤、排班和里程碑规划 |
| `$afrexai-event-management` | 活动现场流程管理、指标监控、ROI 评估和复盘分析 |
| `$activity-campaign-from-ui` | H5 或网页营销活动方案、页面结构和前端原型生成 |
| `$marketing-designer` | 营销创意策略、活动主题、视觉调性和跨渠道传播方案 |

## 我可以帮你做这些

1. 活动整体策划

   调用 `afrexai-event-planner`

   明确活动类型、目标、受众、规模、场地需求和概念提案。

2. 活动执行 SOP

   调用 `campaign-planning`

   拆解筹备期、执行期、收尾期任务，生成责任分工、时间节点和风险预案。

3. 运营预算和供应商管理

   调用 `afrexai-event-planning`

   制定收支预算、供应商清单、后勤保障、人员排班和里程碑计划。

4. 现场管理和效果复盘

   调用 `afrexai-event-management`

   规划现场流程、监控签到率和参与度，输出 ROI 评估和复盘建议。

5. 活动页面和交互方案

   调用 `activity-campaign-from-ui`

   根据活动目标或 UI 参考生成报名页、详情页、倒计时页等营销页面方案。

6. 创意策略和传播计划

   调用 `marketing-designer`

   设计活动主题、创意概念、品牌调性、多版本创意和跨渠道内容排期。

7. 端到端活动交付

   调用 `afrexai-event-planner`、`campaign-planning`、`afrexai-event-planning`、`afrexai-event-management`、`activity-campaign-from-ui`、`marketing-designer`

   从活动概念、执行 SOP、预算供应商、现场管理到页面和传播策略完整串联。

## 完整交付物通常是

```text
活动整体策划方案：活动类型、目标、受众、概念提案

执行 SOP 文档：阶段清单、责任分工、时间节点、风险预案

运营预算与供应商方案：预算明细、供应商清单、后勤保障、排班计划

活动效果分析报告：执行数据、ROI 评估、复盘总结、改进建议

营销活动页面：H5 或网页方案、页面结构、前端原型

创意传播策略：活动主题、视觉风格、跨渠道传播计划
```
