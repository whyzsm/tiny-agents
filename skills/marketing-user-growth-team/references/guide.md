# 用户增长专家团

`$marketing-user-growth-team` 用户增长专家团总入口

## 触发的技能

| 技能 | 能力 |
|---|---|
| `$marketing-user-growth-team` | 用户增长专家团工作流总入口 |
| `$afrexai-growth-engine` | 北极星指标、AARRR+ 增长框架、增长飞轮和实验体系设计 |
| `$cgo` | 获客闭环、渠道组合、增长实验、PLG 和留存系统规划 |
| `$funnel-analyzer` | 转化漏斗创建、流失诊断、行业对标和优化建议 |
| `$customer-onboarding-2` | 新用户入职流程、激活指标、Aha Moment 和摩擦点优化 |
| `$customer-retention` | 客户留存策略、生命周期营销、忠诚度计划和赢回活动 |
| `$afrexai-churn-analyzer` | 流失风险分层、预警阈值和定制化留存干预建议 |

## 我可以帮你做这些

1. 增长体系设计

   调用 `afrexai-growth-engine`

   定义北极星指标、关键增长指标、AARRR+ 框架、增长飞轮和实验优先级。

2. 获客闭环和实验计划

   调用 `cgo`

   设计有机增长、付费增长、病毒式增长的渠道组合，并建立假设、实验、数据、迭代闭环。

3. 转化漏斗诊断

   调用 `funnel-analyzer`

   建立曝光到付费的漏斗，定位流失环节，对标行业基准并输出优化建议。

4. 新用户激活优化

   调用 `customer-onboarding-2`

   设计入职引导、Aha Moment、激活标准和前 30 天流失降低方案。

5. 留存和赢回策略

   调用 `customer-retention`

   构建客户生命周期运营、忠诚度计划、消息触达和休眠用户赢回活动。

6. 流失风险预警

   调用 `afrexai-churn-analyzer`

   根据客户行为和业务数据做风险分层，设置预警阈值并推荐干预动作。

7. 端到端用户增长交付

   调用 `afrexai-growth-engine`、`cgo`、`funnel-analyzer`、`customer-onboarding-2`、`customer-retention`、`afrexai-churn-analyzer`

   先搭建增长体系，再分析获客和漏斗，最后输出激活、留存、赢回和流失预警方案。

## 完整交付物通常是

```text
增长体系框架：北极星指标、AARRR+ 框架、增长飞轮

获客策略与实验计划：渠道策略、实验优先级、预算分配

漏斗分析报告：转化率、流失诊断、行业对标、优化建议

入职与激活方案：用户引导流程、Aha Moment、摩擦消除

留存与赢回策略：忠诚度计划、生命周期营销、赢回活动

流失预警模型：风险分层、干预措施、预警阈值
```
