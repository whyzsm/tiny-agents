# 脚本拆解专家团

`$media-script-breakdown` 脚本拆解专家

## 触发的技能

| 技能 | 能力 |
|---|---|
| `$media-script-breakdown` | 完整脚本拆解工作流总入口 |
| `$seedance2-storyboard-generator` | 四幕结构、剧集拆分、资产提示、Seedance 分镜 |
| `$script-to-storyboard` | 标准分镜表、镜号、景别、画面、人物、场景、音效 |
| `$short-drama-writer` | 竖屏短剧、角色关系、反转节点、对白、场景描述 |
| `$ai-viral-team-script-writing` | 黄金3秒、情绪钩子、槽点埋梗、BGM、互动结尾 |
| `$video-script-generator` | JSON/YAML 分镜、多模板、多平台视频脚本 |
| `$movie` | 多剧集连续生成、人物场景图谱、版本管理、AI 质检 |

## 我可以帮你做这些

1. 四幕剧本生成

   调用 `seedance2-storyboard-generator`

   从故事、文章或创意提炼剧情线，生成四幕结构、剧集拆分、角色场景和资产提示。

2. 剧本转分镜

   调用 `script-to-storyboard`

   把已有剧本解析成标准分镜表，包含镜号、景别、角度、画面、人物、场景和音效。

3. 短剧创作

   调用 `short-drama-writer`

   设计竖屏短剧角色、人物关系、剧情反转、对白和可拍摄场景描述。

4. 短视频情绪钩子

   调用 `ai-viral-team-script-writing`

   生成黄金3秒开头、情绪钩子、槽点埋梗、镜头语言、BGM 和互动结尾。

5. 结构化脚本输出

   调用 `video-script-generator`

   输出 JSON/YAML 分镜数据，多模板适配痛点解决、反转剧情、before-after 等结构。

6. 制片工程管理

   调用 `movie`

   管理多集连续剧情的人物、场景、钩子、版本一致性和质检流程。

## 完整交付物通常是

```text
四幕剧本

标准分镜表格

短剧剧本

短视频分镜和情绪设计

结构化脚本数据

剧本工程文件和制作规划
```
