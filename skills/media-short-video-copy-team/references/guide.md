# 短视频文案专家团

`$media-short-video-copy-team` 短视频文案专家

## 触发的技能

| 技能 | 能力 |
|---|---|
| `$media-short-video-copy-team` | 完整短视频文案工作流总入口 |
| `$douyin-copy-extract` | 提取标题、简介、口播文案、敏感词提醒和时长参考 |
| `$short-video-script-creator` | 根据产品、对标和历史文案批量生成短视频脚本 |
| `$cn-video-script-writer` | 生成多平台选题、口播稿、分镜脚本、标题封面 |
| `$viral-script-writer` | 设计黄金3秒开场、情绪节奏和高留存口播 |
| `$short-video-script` | 一站式生成脚本、标题、封面、拍摄指导和剪辑建议 |
| `$viral-title-generator` | 批量生成高点击标题、改写版本和 AB 测试标题 |

## 我可以帮你做这些

1. 竞品文案提取

   调用 `douyin-copy-extract`

   从公开短视频链接中提取标题、简介、口播文案，生成朗读优化版、精简版和敏感词提醒。

2. 多平台脚本生成

   调用 `short-video-script-creator`

   根据产品信息、内容对标、平台风格和历史文案生成短视频脚本。

3. 平台化文案创作

   调用 `cn-video-script-writer`

   面向抖音、B站、小红书、视频号和 YouTube 生成选题、口播稿、分镜和标题封面。

4. 黄金3秒口播

   调用 `viral-script-writer`

   设计强钩子开场、情绪节奏、反差点和完播率优化脚本。

5. 完整内容包

   调用 `short-video-script`

   输出脚本、标题、封面建议、拍摄指导、剪辑建议和发布准备信息。

6. 爆款标题优化

   调用 `viral-title-generator`

   批量生成多平台标题、AB 测试版本、标题改写和敏感词替换建议。
