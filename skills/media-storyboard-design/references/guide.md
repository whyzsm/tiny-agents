# 分镜设计专家团

`$media-storyboard-design` 分镜设计专家

## 触发的技能

| 技能 | 能力 |
|---|---|
| `$media-storyboard-design` | 完整分镜设计工作流总入口 |
| `$script-to-storyboard` | 剧本转标准分镜表 |
| `$video-cinematography` | 运镜、机位、构图、节奏和转场设计 |
| `$seedance2-video-prompter` | Seedance / 即梦可执行视频提示词 |
| `$prompt-bozo` | AI 绘画与视频画面提示词 |
| `$seedance2-storyboard-generator` | 四幕分镜脚本与资产提示 |
| `$ai-video-script` | 视频脚本、配音文案与字幕 |

## 我可以帮你做这些

1. 剧本转分镜表

   调用 `script-to-storyboard`

   把剧本文本解析成镜号、景别、拍摄角度、画面内容、人物、场景和音效。

2. 电影感镜头运动

   调用 `video-cinematography`

   为每个镜头设计推、拉、摇、移、跟、升、降、构图和转场节奏。

3. Seedance 视频提示词

   调用 `seedance2-video-prompter`

   将创意或分镜转成 Seedance / 即梦可执行的视频生成提示词。

4. AI 画面提示词

   调用 `prompt-bozo`

   参数化生成人物、环境、构图、光影、姿势、风格和生成模式。

5. 四幕分镜脚本

   调用 `seedance2-storyboard-generator`

   从故事或文章生成四幕 AI 视频剧本、角色场景、镜头指令和资产提示。

6. 完整视频素材包

   调用 `ai-video-script`

   输出视频脚本、分镜描述、画面提示词、配音文案和字幕文本。

## 完整交付物通常是

```text
标准分镜表格

镜头运动方案

AI 视频分镜指令

AI 画面提示词

四幕分镜脚本

完整视频素材包
```
