# 视频剪辑专家团

`$media-video-editing` 视频剪辑专家

## 触发的技能

| 技能 | 能力 |
|---|---|
| `$media-video-editing` | 完整视频剪辑工作流总入口 |
| `$ai-video-clipper` | 对话式剪辑、片段合并、背景音乐、粗剪版本 |
| `$video-clip-assistant` | 自动提取高光片段、字幕烧录、短视频导出 |
| `$ffmpeg-video-editor` | 自然语言生成 FFmpeg 命令，处理转码、压缩、裁剪和批量任务 |
| `$video-editor` | 剪切、合并、提取音频、调速、缩放、裁剪、添加字幕 |
| `$ai-agentic-video-editor` | 脚本、分镜、多语言字幕、标题、封面文案 |
| `$remotion-video-toolkit` | Remotion + React 动画视频、模板组件和批量渲染 |

## 我可以帮你做这些

1. AI 初剪

   调用 `ai-video-clipper`

   通过自然语言完成片段剪切、合并、背景音乐添加和粗剪整理。

2. 精彩片段提取

   调用 `video-clip-assistant`

   自动提取高光片段，生成字幕并烧录，导出适配短视频平台的版本。

3. FFmpeg 专业处理

   调用 `ffmpeg-video-editor`

   将自然语言需求转换为 FFmpeg 命令，执行转码、压缩、裁剪、合并和批处理。

4. 综合视频编辑

   调用 `video-editor`

   完成剪切、合并、格式转换、提取音频、添加字幕、缩放、裁剪和调速。

5. 文案字幕生成

   调用 `ai-agentic-video-editor`

   生成视频脚本、分镜、多语言字幕、标题、简介和封面文案。

6. 程序化视频创作

   调用 `remotion-video-toolkit`

   用 Remotion + React 制作动画视频、字幕特效、图表视频和可复用模板。
