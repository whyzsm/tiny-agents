# Nursing Theory Matcher v4

## 本 skill 的定位
- 面向护理学硕博论文、临床护理研究、护理教育研究、护理管理与实施研究。
- 输出：主理论 + 备选理论 + 理由 + 变量/维度映射 + 理论依据段落 + 不选其他理论的理由。
