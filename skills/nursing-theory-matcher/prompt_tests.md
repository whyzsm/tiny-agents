# Nursing Theory Matcher v4 Prompt Tests

本文件用于对 skill 做回归测试。建议每个测试都观察以下几点：
1. 是否选对主理论
2. 是否能给出合理的辅理论/备选理论
3. 是否能说明为什么不是其他理论
4. 是否能把理论概念转成变量、访谈维度或干预模块

## 症状自我管理

### S01 老年慢性心衰患者出院后症状负担与自我管理行为的关系研究
**测试提示词：**
```text
请根据以下护理研究选题匹配理论基础/理论框架，并输出：研究画像、首选理论、备选理论、变量或访谈维度映射、为什么不是其他理论、可直接写入开题的理论依据段落。
题目：老年慢性心衰患者出院后症状负担与自我管理行为的关系研究
对象：老年慢性心衰患者
设计：横断面研究
核心现象：症状负担、自我管理、出院后管理
```
**期望主理论：** `ssmf_chc`
**期望辅/备选：** self_efficacy, symptom_management_theory, self_care_chronic_illness
**应避免优先推荐：** pender_hpm
**判断理由：** 复杂慢病 + 症状 + 自我管理是SSMF-CHC典型场景。
