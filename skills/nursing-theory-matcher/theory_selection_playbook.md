# 护理研究理论选型手册（Theory Selection Playbook）

> 目标：帮助护理研究者在开题、论文写作、选题论证、研究设计和投稿修改时，快速判断“**什么题目适合什么理论**”。

## 一、先分清：你是在选“方法论”还是“理论框架”？
### 方法论适合：探索经历、解释过程、建构理论、描绘文化/故事/经验本质
- 扎根理论：研究“过程”“互动”“形成机制”
- 现象学：研究“生活经验本质”
- 解释现象学：研究“体验意义”
- 民族志：研究“文化/群体实践”
- 叙事研究：研究“疾病故事与身份建构”
- 个案研究：研究“复杂个案/项目/病区”

### 理论框架/模型适合：定量、混合方法中的量化主框架、干预研究、实施/质量改进研究

## 二、按研究核心现象选理论
### A. 症状 + 自我管理
- 首选：SSMF-CHC、SMT、TOUS、SC-CI
- 辅理论：自我效能理论
- 典型误用：症状题目只用泛化健康促进模型

### B. 依从性 / 风险感知 / 预防行为 / 健康教育
- 首选：PMT、HBM、TPB、自我效能、Pender HPM、TTM、IMB

### C. 家庭参与 / 照护者 / 共同管理
- 首选：SFMF、IFSMT、FCC、照护科学框架
- 辅理论：社会支持、韧性

### D. 出院转衔 / 角色变化 / 过渡期 / 不确定感
- 首选：Meleis 转变理论、Mishel 疾病不确定性理论、Roy 适应模式
- 妇幼特异场景：Mercer 母亲角色获得理论

### E. 关怀 / 舒适 / 人际关系 / 护患互动
- 首选：Watson、Kolcaba、Swanson、Peplau、Travelbee

### F. 文化 / 少数族裔 / 多元背景护理
- 首选：Leininger 跨文化护理理论
- 若为文化质性研究：民族志

### G. 实施科学 / 指南落实 / 证据转化 / 项目落地 / 质量改进
- 首选：COM-B、TDF、BCW、CFIR、RE-AIM、PARIHS / i-PARIHS、KTA、NPT、MRC、DOI

### H. 护理质量 / 卫生服务利用 / 患者报告结局 / 共同决策
- 首选：Donabedian、Andersen、Wilson-Cleary、PCC、FCC、SDM

## 三、常见双推荐
- 症状 + 行为依从性：SSMF-CHC/SMT + 自我效能或PMT
- 家庭 + 症状自我管理：SFMF + SSMF-CHC
- 转衔 + 不确定感：Meleis + Mishel
- 实施 + 项目评价：COM-B/CFIR + RE-AIM
- 患者中心 + 共同决策：PCC + SDM
- 关怀 + 舒适：Watson + Kolcaba

## 四、常见错误清单
1. 质性研究不区分方法论与理论框架
2. 症状题目只推荐泛化行为模型
3. 家庭/照护者题目忽略家庭层框架
4. 转衔题目遗漏 Meleis
5. 不确定感题目遗漏 Mishel
6. 实施研究不用实施科学框架
7. 只报理论名称，不做变量映射
8. 没有解释为什么不用其他理论
