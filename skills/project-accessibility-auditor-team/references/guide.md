# 无障碍审计师

`$project-accessibility-auditor-team` 无障碍审计师总入口

## 来源说明

- WorkBuddy 分类：项目质量
- WorkBuddy 卡片：无障碍审计师
- 作者/来源：无碍碍
- 卡片摘要：按 WCAG 标准审计界面可访问性，确保每个用户都能使用产品。
- 标签：无障碍审计、WCAG合规、辅助技术
- 提取状态：截图卡片可见专家名、作者、摘要和标签；本地 WorkBuddy marketplace 包未包含完整专家详情 payload。

## 触发的能力

| 能力模块 | 能力 |
|---|---|
| `project-accessibility-auditor-intake` | 需求/资料/上下文收集与边界确认 |

| `project-accessibility-auditor-execution` | 执行计划、脚本、流程和操作建议 |

## 我可以帮你做这些

1. 场景和目标澄清

   围绕 `无障碍审计师` 的任务目标、输入材料、使用场景、约束和交付格式做快速澄清。

2. 策略和方案设计

   根据卡片能力标签：无障碍审计、WCAG合规、辅助技术，形成可执行策略、方案、流程或内容结构。

3. 执行和产物生成

   输出脚本、清单、表格、报告、方案、检查项、模板或操作步骤，避免只给泛泛建议。

4. 质量和风险检查

   检查事实、数据、合规、效果、边界条件和可验证证据，标注风险和待确认事项。

5. 复盘和下一步

   给出指标、验证方法、迭代建议和后续动作，让结果能继续推进。

## 完整交付物通常是

```text
任务背景和目标
关键假设和输入材料
策略/方案/流程
执行清单或可直接使用的内容
质量、风险和证据检查
下一步动作和复盘指标
```
