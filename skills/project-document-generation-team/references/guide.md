## 触发的能力

| 能力模块 | 能力 |
|---|---|
| `document-research` | 背景调研、资料归纳、事实核验和素材库 |
| `outline-planning` | 文档目标、读者、结构、大纲和章节规划 |
| `content-writing` | 长文档撰写、语言风格、图表说明和摘要 |
| `spec-normalization` | 格式、编号、术语、引用、模板和规格统一 |
| `compliance-audit` | 合规、敏感信息、风险表述和审计记录 |
| `review-loop` | 审稿意见、修订记录、版本管理和交付清单 |

## 与已有专家团的边界

| 已有专家团 | 主要关系 |
|---|---|
| `marketing-official-document-writing` | 偏公文写作；本团偏企业级长文档、规格和审计。 |
| `academic-literature-review` | 偏学术综述；本团覆盖项目/企业文档交付。 |

## 我可以帮你做这些

1. 文档目标

   明确读者、用途、范围、合规边界和交付格式。

2. 深度调研

   整理资料、证据、引用、事实和缺口。

3. 大纲规划

   输出结构、章节、重点、信息层级和写作节奏。

4. 撰写规范

   生成正文、摘要、图表说明、术语和格式。

5. 审计交付

   完成合规审核、修订循环、版本记录和交付清单。

## 完整交付物通常是

```text
资料摘要
文档大纲
长文档正文
格式规范
合规审核表
修订记录
```
