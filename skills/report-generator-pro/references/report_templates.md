## 日报模板（daily_default.md）

```markdown
# {姓名}{部门} {日期} 日报

## ✅ 今日完成
- [主要工作项1：成果描述]
- [主要工作项2：成果描述]
- [临时任务/突发事件处理]

## 🔄 进行中
- [工作项]：当前进度约 {百分比}%

## ⚠️ 待解决
- [问题/障碍描述]（计划处理方式：xxx）

## 📋 明日计划
1. [优先任务1]
2. [优先任务2]

---
*日报自动生成 · {生成时间}*
```

**字数要求**：300～500 字
**核心原则**：具体、有数据、不写空话

## HTML 邮件正文模板规范

邮件正文使用 Jinja2 风格 HTML 模板，核心规范：

1. **响应式设计**：适配手机/桌面邮件客户端
2. **品牌色**：主色 `#1E88E5`（蓝色），强调色 `#4CAF50`（绿色）
3. **字体**：中文字体优先：微软雅黑、PingFang SC、无衬线
4. **结构**：
```html
<!DOCTYPE html>
<html>
<head><meta charset="utf-8"><meta name="viewport" content="width=device-width,initial-scale=1">
<title>{{subject}}</title></head>
<body style="font-family: 'Microsoft YaHei', sans-serif; background:#f5f5f5; margin:0; padding:0;">
  <table width="100%" cellpadding="0" cellspacing="0">
    <tr><td align="center">
      <table width="600" cellpadding="0" cellspacing="0" style="background:white; border-radius:8px;">
        <!-- 头部 -->
        <tr><td style="background:#1E88E5; color:white; padding:20px; border-radius:8px 8px 0 0;">
          <h2 style="margin:0;">{{report_title}}</h2>
          <p style="margin:5px 0 0;">{{period}}</p>
        </td></tr>
        <!-- 正文内容 -->
        <tr><td style="padding:20px;">{{content_html}}</td></tr>
        <!-- 底部 -->
        <tr><td style="padding:15px; border-top:1px solid #eee; color:#999; font-size:12px;">
          {{signature}}<br>自动生成于 {{generate_time}}
        </td></tr>
      </table>
    </td></tr>
  </table>
</body>
</html>
```

5. **兼容性注意**：
   - 不使用外部 CSS 文件
   - 不使用 JavaScript
   - 核心样式用 inline style
   - 图片使用 HTTPS 链接或 Base64 内嵌
