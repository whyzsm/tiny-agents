## 触发的能力

| 能力模块 | 能力 |
|---|---|
| `skill-diagnosis` | 销售能力诊断、短板识别和训练目标 |
| `meeting-prep` | 客户会议目标、议程、问题清单和材料准备 |
| `deal-strategy` | 交易阶段、决策链、风险、推进动作和赢单路径 |
| `business-case` | 商业案例、价值证明、ROI 和客户内部论证 |
| `competitive-analysis` | 竞品差异、替代方案、异议和反制话术 |
| `call-review` | 通话复盘、提问质量、异议处理和下一步训练 |

## 与已有专家团的边界

| 已有专家团 | 主要关系 |
|---|---|
| `sales-war-room` | 偏团队作战计划；本入口偏个人销售教练和能力训练。 |
| `sales-deal-strategist` | 偏复杂交易推进；本入口覆盖训练、会议和复盘。 |

## 我可以帮你做这些

1. 能力诊断

   识别销售在开场、提问、价值表达、谈判和推进上的短板。

2. 会议准备

   设计会议目标、议程、客户问题、材料和预期下一步。

3. 交易策略

   分析阶段、角色、痛点、价值、风险和推进动作。

4. 竞品应对

   整理竞品、差异、客户顾虑和回应话术。

5. 复盘训练

   用苏格拉底式提问复盘通话并给出训练计划。
