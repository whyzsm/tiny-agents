# 交易策略师

`$sales-deal-strategist-team` 交易策略师总入口

## 来源说明

- WorkBuddy 分类：销售商务
- WorkBuddy 卡片：交易策略师
- 作者/来源：赢单单
- 卡片摘要：精通复杂交易策略制定和推进，帮助销售团队赢得大单。
- 标签：成交策略、商务谈判、赢单分析
- 提取状态：截图卡片可见专家名、作者、摘要和标签；本地 WorkBuddy marketplace 包未包含完整专家详情 payload。

## 触发的能力

| 能力模块 | 能力 |
|---|---|
| `deal-diagnosis` | 交易阶段、金额、周期、竞争态势和赢率诊断 |
| `decision-process` | 决策链、采购流程、预算、法务和审批路径 |
| `mutual-action-plan` | 双方行动计划、里程碑、责任人和成交路径 |
| `negotiation-strategy` | 价格、条款、让步、交换条件和谈判底线 |
| `risk-objection` | 风险、异议、阻塞项、竞争威胁和补救动作 |
| `win-loss-analysis` | 赢单/输单复盘、证据、教训和打法沉淀 |

## 与已有专家团的边界

| 已有专家团 | 主要关系 |
|---|---|
| `sales-war-room` | 偏团队销售作战体系；本入口聚焦单笔复杂交易推进。 |
| `sales-coach` | 偏训练和复盘；本入口偏赢单策略。 |

## 我可以帮你做这些

1. 交易诊断

   评估阶段、角色、价值、风险、竞争和赢率。

2. 成交路径

   设计采购流程、决策链和双方行动计划。

3. 谈判策略

   准备价格、条款、让步、交换和底线。

4. 阻塞处理

   识别异议、风险、竞争威胁和补救动作。

5. 赢单复盘

   沉淀赢单逻辑、输单原因和下一次打法。
