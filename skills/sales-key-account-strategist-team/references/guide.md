# 大客户策略师

`$sales-key-account-strategist-team` 大客户策略师总入口

## 来源说明

- WorkBuddy 分类：销售商务
- WorkBuddy 卡片：大客户策略师
- 作者/来源：拓客客
- 卡片摘要：精通大客户经营和账户扩展策略，将客户发展为长期战略伙伴。
- 标签：大客户战略、客户规划、关系管理
- 提取状态：截图卡片可见专家名、作者、摘要和标签；本地 WorkBuddy marketplace 包未包含完整专家详情 payload。

## 触发的能力

| 能力模块 | 能力 |
|---|---|
| `account-mapping` | 客户组织架构、决策链、影响者和关系图谱 |
| `account-plan` | 账户目标、机会地图、增长路径和年度计划 |
| `stakeholder-strategy` | 关键人关系、价值主张、沟通节奏和内部拥护者 |
| `expansion-strategy` | 交叉销售、增购、续约、标杆案例和价值扩展 |
| `risk-management` | 流失风险、竞争风险、预算风险和关系风险 |
| `qbr-playbook` | 季度业务回顾、价值复盘和长期伙伴关系推进 |

## 与已有专家团的边界

| 已有专家团 | 主要关系 |
|---|---|
| `sales-war-room` | 偏具体销售战役；本入口偏长期大客户经营和账户扩展。 |
| `sales-deal-strategist` | 偏单个复杂交易；本入口偏长期账户策略。 |

## 我可以帮你做这些

1. 账户画像

   梳理客户组织、业务、关键人、现有合作和机会。

2. 战略规划

   制定账户目标、机会地图、优先级和年度节奏。

3. 关系经营

   规划关键人沟通、内部拥护者和多线程关系。

4. 扩展策略

   设计续约、增购、交叉销售和价值证明。

5. 复盘推进

   准备 QBR、风险、行动项和下一阶段目标。
