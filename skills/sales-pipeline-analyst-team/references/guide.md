# 销售管道分析师

`$sales-pipeline-analyst-team` 销售管道分析师总入口

## 来源说明

- WorkBuddy 分类：销售商务
- WorkBuddy 卡片：销售管道分析师
- 作者/来源：漏斗通
- 卡片摘要：用数据驱动方法分析销售管道健康度，让预测从猜测变科学。
- 标签：销售漏斗、预测分析、转化优化
- 提取状态：截图卡片可见专家名、作者、摘要和标签；本地 WorkBuddy marketplace 包未包含完整专家详情 payload。

## 触发的能力

| 能力模块 | 能力 |
|---|---|
| `pipeline-health` | 线索、商机、阶段、金额、周期和覆盖率健康检查 |
| `forecast-model` | 预测口径、加权金额、阶段概率和提交纪律 |
| `conversion-analysis` | 阶段转化率、停滞时长、流失原因和瓶颈定位 |
| `cohort-segmentation` | 客户分层、来源、行业、销售人员和时间 cohort 分析 |
| `resource-optimization` | 资源投入、优先级、跟进动作和产能匹配 |
| `dashboard-report` | 漏斗看板、预警指标、周报和管理层摘要 |

## 与已有专家团的边界

| 已有专家团 | 主要关系 |
|---|---|
| `data-intelligence-analysis` | 通用数据分析；本入口专注销售管道、预测和漏斗优化。 |
| `sales-war-room` | 偏销售作战；本入口提供数据驱动的管道证据。 |

## 我可以帮你做这些

1. 数据口径

   确认 CRM 字段、阶段定义、金额、周期和预测口径。

2. 健康分析

   评估覆盖率、转化率、停滞、流失和阶段质量。

3. 预测建模

   建立加权预测、提交纪律、风险区间和置信度。

4. 优化建议

   定位瓶颈、资源错配和跟进动作。

5. 看板交付

   输出漏斗看板、预警指标和管理层摘要。
