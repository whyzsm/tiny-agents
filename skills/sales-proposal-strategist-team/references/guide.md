# 方案策划师

`$sales-proposal-strategist-team` 方案策划师总入口

## 来源说明

- WorkBuddy 分类：销售商务
- WorkBuddy 卡片：方案策划师
- 作者/来源：策必中
- 卡片摘要：精通销售提案和方案设计，将价值转化为无法拒绝的商业论证。
- 标签：方案撰写、投标策略、RFP响应
- 提取状态：截图卡片可见专家名、作者、摘要和标签；本地 WorkBuddy marketplace 包未包含完整专家详情 payload。

## 触发的能力

| 能力模块 | 能力 |
|---|---|
| `proposal-brief` | 客户背景、目标、痛点、约束和决策标准梳理 |
| `value-argument` | 价值主张、商业论证、ROI 和差异化表达 |
| `solution-structure` | 方案结构、章节、大纲、附件和故事线 |
| `rfp-response` | RFP 条款拆解、响应矩阵、符合性和偏离说明 |
| `bid-strategy` | 投标策略、评分项、竞品对位和风险控制 |
| `proposal-review` | 方案审阅、漏洞补齐、说服力和一致性检查 |

## 与已有专家团的边界

| 已有专家团 | 主要关系 |
|---|---|
| `project-document-generation` | 偏企业长文档生成；本入口聚焦销售提案、投标和 RFP。 |
| `sales-presales-technical-consultant` | 偏技术方案讲解；本入口偏提案和商业论证。 |

## 我可以帮你做这些

1. 提案诊断

   理解客户目标、采购要求、评分标准、竞争态势和交付边界。

2. 价值论证

   把功能转成业务价值、ROI、风险降低和管理层语言。

3. 方案成稿

   设计提案结构、故事线、图表和核心页面。

4. RFP 响应

   拆解招标条款、生成响应矩阵和偏离说明。

5. 投标优化

   围绕评分项、风险和竞品优化赢标概率。
