# 期末评语模板集

> 包含多种风格的评语模板，适用于不同类型学生。

## 【标准版】通用评语

```
你是一个XXXX的孩子。（整体定性）

在学习上，你XXXX。（学习态度+成绩表现，用具体事例更佳）
在生活中，你XXXX。（品德/劳动/体育/审美，1-2条即可）

特别值得肯定的是XXXX。（最大亮点，1-2条）

老师希望你在XXXX方面继续努力，（具体改进建议）
相信你一定可以XXXX。（鼓励结尾，展望未来）

—— 班主任 XXX
```

## 【优点突出型】用于表现优秀学生

```
你是一个XXXXXXXX的孩子。（优点定性）

这学期，你XXXXXXXX。（用具体事例佐证优点）
XXXXXXXX。（另一项优点）

老师为你感到骄傲！
希望你在XXXX方面继续突破，
相信你一定可以XXXXXXXX！

—— 班主任 XXX
```

## 【鼓励进步型】用于有进步的学生

```
老师欣喜地看到，这学期你XXXXXXXX。（进步点）
（举例说明进步，如：上课举手发言更积极了、作业完成率提高了等）

你是一个XXXXXXXX的孩子。（本性定性）
XXXXXXXX。（继续肯定）

老师相信，只要你XXXXXXXX，（保持/坚持某项习惯）
你一定会XXXXXXXX！

—— 班主任 XXX
```

## 【温和建议型】用于需要改进的学生

```
你是一个XXXXXXXX的孩子。（正面定性，避免负面标签）

这学期，老师看到你在XXXX方面的努力。（找到哪怕一点点优点）
不过，XXXXXXXX。（温和指出问题，注意措辞正向）

老师希望XXXXXXXX，（具体建议，1条即可）
我们一起努力，好吗？（表达支持和期待）

—— 班主任 XXX
```

## 【多才多艺型】用于特长学生

```
你是一个XXXXXXXX的孩子。（性格定性）

这学期，你在XXXXXXXX方面表现突出，
XXXXXXXX。（特长表现+具体事例）

老师希望你在保持特长的同时，
也能XXXXXXXX，（平衡发展的建议）
相信你一定可以XXXX！

—— 班主任 XXX
```

## 评语写作规范

| 要求 | 说明 |
|------|------|
| 字数 | 100-200字为宜 |
| 语气 | 真诚、鼓励、具体 |
| 禁忌 | 避免"差生""笨蛋""不听话"等负面词 |
| 突出 | 每条评语突出1-2个亮点 |
| 改进 | 每次评语最多提1个改进建议 |
| 署名 | 注明班主任姓名 |
