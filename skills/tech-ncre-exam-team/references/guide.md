## 触发的能力

| 能力模块 | 能力 |
|---|---|
| `ncre-level-planning` | NCRE 一级到四级考试路径、科目选择和备考计划 |
| `office-practice` | Office 操作题、函数、排版、演示和数据处理训练 |
| `programming-practice` | C/C++/Java/Python 等程序设计知识点和练习 |
| `database-practice` | 数据库原理、SQL、Access/MySQL 题型和实操 |
| `network-security-practice` | 网络基础、安全、操作系统和计算机基础知识 |
| `mock-exam-grading` | 模拟卷、错题归因、评分建议和复习计划 |

## 与已有专家团的边界

| 已有专家团 | 主要关系 |

| `education-quiz-generation` | 负责通用题库生成；本团聚焦 NCRE 考纲、等级、题型和备考路径。 |

| `tech-software-development` | 负责软件开发交付；本团面向考试学习，不承担工程交付。 |

## 我可以帮你做这些

1. 等级和科目规划

   根据目标等级、基础、时间和考试区域制定备考路线。

2. 知识点梳理

   按 Office、编程、数据库、网络安全和基础知识拆解考点。

3. 练习与讲解

   生成专项练习、操作题步骤、代码题思路和答案解析。

4. 错题诊断

   分析错因、知识漏洞、题型弱点和复习优先级。

5. 模拟考试

   组织模拟卷、评分建议、考前清单和冲刺计划。

## 完整交付物通常是

```text
备考路径
知识点清单
专项练习
错题分析
模拟卷和评分
冲刺计划
```
