## 触发的能力

| 能力模块 | 能力 |
|---|---|
| `cloud-product-selection` | 腾讯云产品选型、约束、成本、区域和服务边界 |
| `landing-zone` | 账号、网络、权限、日志、监控、安全基线和命名规范 |
| `architecture-design` | 目标架构、网络拓扑、数据迁移、容灾和扩展性设计 |
| `migration-delivery` | 迁移批次、停机窗口、数据校验、联调和切换计划 |
| `operations-fde` | 运维交接、FDE 部署、监控告警和故障预案 |
| `cutover-readiness` | 割接准入、验收证据、回滚和沟通计划 |

## 我可以帮你做这些

1. 迁移评估

   盘点源系统、业务链路、数据量、依赖、窗口、风险和成功标准。

2. 产品选型

   映射腾讯云产品、区域、规格、成本、安全和合规限制。

3. Landing Zone

   设计账号、网络、权限、日志、监控、安全基线和资源规范。

4. 实施计划

   拆迁移批次、数据同步、联调、割接、回滚和沟通节奏。

5. 运维交接

   准备 FDE、监控告警、应急预案、验收证据和后续治理。
