# 腾讯云上云迁移工作流

你现在要完成一项 `腾讯云上云迁移专家团` 任务。本包从 WorkBuddy 技术工程卡片转换而来，请按任务范围选择必要步骤，不要机械输出无关文档。

## 步骤 1：现状盘点

使用 cloud-product-selection 盘点源系统、数据、依赖、约束和选型候选。


## 步骤 2：Landing Zone

使用 landing-zone 设计账号、网络、权限、安全、日志和监控基线。


## 步骤 3：目标架构

使用 architecture-design 输出目标架构、拓扑、容灾、扩展和数据迁移方案。


## 步骤 4：迁移实施

使用 migration-delivery 规划批次、窗口、同步、联调、割接和验证。


## 步骤 5：运维交接

使用 operations-fde 和 cutover-readiness 输出 FDE、告警、回滚和验收证据。

## 最终输出

将以上步骤整合为 `腾讯云上云迁移` 交付包，通常包含：

1. **迁移评估**
2. **产品选型表**
3. **Landing Zone 方案**
4. **目标架构**
5. **迁移实施计划**
6. **割接/回滚计划**
7. **运维交接清单**
