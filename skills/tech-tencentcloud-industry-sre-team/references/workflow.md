# 腾讯云行业 SRE工作流

你现在要完成一项 `腾讯云行业 SRE 专家团` 任务。本包从 WorkBuddy 技术工程卡片转换而来，请按任务范围选择必要步骤，不要机械输出无关文档。

## 步骤 1：行业识别

使用 industry-sre-playbook 判断行业、业务峰值、核心链路和主要可靠性指标。


## 步骤 2：云巡检

使用 cloud-inspection 梳理云资源、网络、权限、证书、监控、备份和成本风险。


## 步骤 3：架构治理

使用 architecture-governance 评估高可用、容灾、容量、隔离和安全治理。


## 步骤 4：事件响应

使用 incident-response 设计故障分级、告警、值班、应急和复盘流程。


## 步骤 5：SRE 路线图

使用 service-reliability 和 industry-runbook 输出 SLO、演练、改造计划和 runbook。

## 最终输出

将以上步骤整合为 `腾讯云行业 SRE` 交付包，通常包含：

1. **行业 SRE 画像**
2. **云巡检报告**
3. **架构治理建议**
4. **SLO/SLI 指标**
5. **事件响应 runbook**
6. **治理路线图**
