## 我可以帮你做这些

1. SDK 接入

   规划 aegis SDK 平台、初始化参数、用户标识、采样、环境和版本信息。

2. 性能指标

   分析 WebVitals、页面加载、资源瀑布、接口耗时和体验瓶颈。

3. 异常诊断

   聚合 JS 异常、白屏、Promise、资源加载和接口错误。

4. 链路联动

   关联 RUM、APM、日志、trace 和后端接口，定位端到端问题。

5. 观测报告

   输出指标基线、告警阈值、优化建议、验证计划和复盘。
