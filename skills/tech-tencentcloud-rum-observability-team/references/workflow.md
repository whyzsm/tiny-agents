# 腾讯云 RUM 全链路工作流

你现在要完成一项 `腾讯云 RUM 全链路专家团` 任务。本包从 WorkBuddy 技术工程卡片转换而来，请按任务范围选择必要步骤，不要机械输出无关文档。

## 步骤 1：接入设计

使用 aegis-sdk-integration 确定平台、环境、版本、用户标识、采样和隐私边界。


## 步骤 2：WebVitals 分析

使用 web-vitals-analysis 建立 LCP、FCP、CLS、INP/TTFB 等体验指标基线。


## 步骤 3：异常和资源分析

使用 exception-monitoring 和 api-resource-analysis 聚合错误、接口和资源瓶颈。


## 步骤 4：RUM-APM 联动

使用 rum-apm-correlation 建立 trace、接口、日志和用户会话的关联路径。


## 步骤 5：报告和优化

使用 observability-report 输出告警阈值、优化建议、验证计划和复盘模板。

## 最终输出

将以上步骤整合为 `腾讯云 RUM 全链路` 交付包，通常包含：

1. **aegis SDK 接入方案**
2. **WebVitals 报告**
3. **异常分析**
4. **接口/资源分析**
5. **RUM-APM 联动方案**
6. **性能优化建议**
