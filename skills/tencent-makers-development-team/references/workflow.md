# Makers 开发工作流

你现在要完成一项 `Makers 开发专家团` 任务。本包从 WorkBuddy `腾讯专区` 卡片转换而来，请按任务范围选择必要步骤，不要机械输出无关文档。

## 步骤 1：需求到架构

使用相关能力完成：拆分前端、后端、Agent、数据和部署边界。


## 步骤 2：应用实现

使用相关能力完成：规划组件、API、鉴权、数据流和 Agent 能力。


## 步骤 3：EdgeOne 部署

使用相关能力完成：配置构建、环境、Serverless、域名和全球加速。


## 步骤 4：联调验证

使用相关能力完成：检查接口、页面、Agent 会话、日志和访问体验。


## 步骤 5：交付收口

使用相关能力完成：输出部署地址、验证证据、风险和后续优化。

## 最终输出

将以上步骤整合为 `Makers 开发` 交付包，通常包含：

1. **应用方案**
2. **前后端任务拆解**
3. **AI Agent 接入方案**
4. **EdgeOne 部署步骤**
5. **验证清单**
6. **交付说明**
