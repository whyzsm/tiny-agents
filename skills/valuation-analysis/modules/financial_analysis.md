# 财务分析技能模块

## 核心框架：《股市真规则》六步财务分析

### 一、成长性分析

**收入质量检验**
```
健康的收入增长 = 销售增长 > 存货增长 > 应收账款增长
```

**危险信号**：
- 应收账款增速 > 收入增速（赊销推动增长）
- 存货增速 > 收入增速（滞销风险）
- 一次性收入占比过高（不可持续）

**关键指标**：
| 指标 | 计算方法 | 健康标准 |
|------|---------|---------|
| 销售增长率 | (本年-上年)/上年 | 稳定 >10% |
| 递延收入增长率 | 负债端递延收入变化 | 增长 = 未来收入保障 |
| 单位经济模型 | 收入/用户数/单客价值 | 持续提升 |

---

### 二、盈利能力分析

**利润质量检验**
```
真实盈利能力 = 经营现金流 / 净利润 ≈ 1.0+
```

**危险信号**：
- 净利润↑但经营现金流↓（利润造假或营运资本恶化）
- 毛利率持续下降（竞争加剧或定价权丧失）
- 非经常性收益占比高（不可持续）

**核心指标**：
| 指标 | 优秀 | 一般 | 危险 |
|------|-----|------|------|
| 毛利率 | >40% | 20-40% | <20% |
| 营业利润率 | >15% | 5-15% | <5% |
| 净利润率 | >10% | 3-10% | <3% |
| FCF/销售额 | >5% | 0-5% | <0% |

---

### 三、财务健康状况检查

#### 3.1 流动性检查
```
流动比率 = 流动资产 / 流动负债 > 1.5
速动比率 = (流动资产-存货) / 流动负债 > 1.0
```

#### 3.2 杠杆检查
```
负债/权益比 = 总负债 / 股东权益 < 1.0 (保守)
已获利息倍数 = EBIT / 利息支出 > 5倍
自由现金流/总负债 > 10%
```

#### 3.3 营运资本检查
```
应收账款周转天数 = 平均应收账款 / (年收入/365) - 趋势下降为佳
存货周转天数 = 平均存货 / (年销售成本/365) - 行业对比
应付账款周转天数 - 稳定或延长（不占供应商便宜）
```

---

### 四、六个危险信号（必须逐项排除）

| # | 危险信号 | 检查方法 | 危险阈值 |
|---|---------|---------|---------|
| 1 | 现金流与利润背离 | 经营现金流 vs 净利润趋势 | 净利润↑但经营现金流↓ |
| 2 | 连续非经常性费用 | 利润表"其他"/"非经常性"项目 | 3年内≥2次 |
| 3 | 连续收购 | 现金流量表投资活动 | 每年大额收购 |
| 4 | 高管/审计师离职 | 年报管理层变更 | 无解释离职 |
| 5 | 应收账款异常 | 应收增速 vs 收入增速 | 应收增速 > 收入增速 |
| 6 | 赊销条件变更 | 10-K/年报脚注 | 放宽信用条款 |

**检查清单**：
```
□ 经营现金流/净利润 连续3年 > 0.8
□ 毛利率 连续5年稳定或上升
□ 应收账款周转天数 连续3年稳定或下降
□ 负债/权益比 < 1.5
□ 无大额商誉减值风险
□ CFO未频繁变更
```

---

### 五、盈利持续性评估

**ROE杜邦分析**
```
ROE = 净利润率 × 资产周转率 × 权益乘数
```

**优质ROE来源**：
- ✅ 高净利润率（定价权）+ 正常周转
- ✅ 正常利润率 + 高周转（零售/超市）
- ❌ 高杠杆支撑的ROE（风险高）

**历史稳定性检查**：
```
□ ROE 连续10年 > 15%
□ ROA 连续10年 > 6%
□ 毛利率 连续10年稳定
□ 净利润率 连续10年 > 10%
```

---

### 六、会计操纵识别

**红旗信号**：
1. **收入确认时点异常** - 季末突击收入
2. **费用资本化** - 本应费用化的支出资本化
3. **商誉膨胀** - 大额收购产生高额商誉
4. **关联交易** - 与关联方大量交易
5. **审计意见非标** - 保留意见或无法表示意见

**核查方法**：
```
□ 阅读审计报告"关键审计事项"
□ 对比同行会计政策差异
□ 检查应收账款账龄结构
□ 分析存货跌价准备计提合理性
□ 核查资本化支出合理性
```

## 快速分析工作流

```
Step 1: 成长性检查（5分钟）
   └─ 收入增长率、递延收入、增长来源

Step 2: 盈利能力检查（5分钟）
   └─ 毛利率、净利润率、FCF/销售额、ROE

Step 3: 财务健康检查（5分钟）
   └─ 流动比率、负债/权益、六个危险信号

Step 4: 盈利持续性评估（5分钟）
   └─ 10年ROE趋势、ROE来源分析

Step 5: 会计操纵筛查（5分钟）
   └─ 审计意见、关联交易、收入确认

Total: 25分钟完成基础财务分析
```
