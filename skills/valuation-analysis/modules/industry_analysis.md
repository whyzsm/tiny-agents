# 行业分析技能模块

## 核心框架：《股市真规则》行业分析方法

### 一、行业吸引力评估（五力模型简化版）

```
行业吸引力 = f(增长性, 盈利性, 稳定性, 竞争格局)
```

**评估矩阵**：
| 维度 | 优秀行业 | 一般行业 | 糟糕行业 |
|------|---------|---------|---------|
| **增长率** | >GDP增速2倍 | ≈GDP增速 | <GDP增速 |
| **平均ROIC** | >15% | 8-15% | <8% |
| **进入壁垒** | 高 | 中等 | 低 |
| **替代品威胁** | 低 | 中等 | 高 |
| **定价权** | 强 | 一般 | 无 |

---

### 二、行业生命周期定位

```
导入期 → 成长期 → 成熟期 → 衰退期
  |        |        |        |
高风险   高成长   稳定现金  价值陷阱
高亏损   高投入   股息回报  避免进入
```

**投资策略匹配**：
| 生命周期 | 估值重点 | 关键指标 | 典型行业 |
|---------|---------|---------|---------|
| **导入期** | 市场份额、用户增长 | DAU/MAU、获客成本 | AI、元宇宙 |
| **成长期** | 收入增长、利润率趋势 | 收入增速、毛利率 | 新能源车、云计算 |
| **成熟期** | 现金流、ROIC、股息 | FCF、ROE、分红率 | 白酒、银行 |
| **衰退期** | 清算价值、现金流 | 资产价值、经营现金流 | 传统纸媒、煤炭 |

### 三、行业特定分析框架

#### 1. 软件/SaaS行业

**关键指标**：
```
□ 月度经常性收入(MRR) / 年度经常性收入(ARR)
□ 净收入留存率(Net Dollar Retention) > 100%
□ 获客成本(CAC) vs 客户终身价值(LTV) - LTV/CAC > 3
□ 付费用户增长
□ 递延收入（未来收入保障）
```

**护城河来源**：
- 高转换成本（客户培训、数据迁移）
- 网络效应（平台型）
- 品牌/专业性（Adobe、Intuit）

---

#### 2. 硬件/半导体行业

**关键指标**：
```
□ 存货周转率（越高越好）
□ 资本支出/收入（警惕过高）
□ 研发费用率（技术领先指标）
□ 产能利用率
□ 客户集中度（前5大客户占比）
```

**周期性判断**：
```
扩张期：收入快速增长 → 毛利率提升 → 加产能
衰退期：收入下滑 → 固定成本压力 → 去库存

买入时机：周期低谷，PE高但即将反转
卖出时机：周期顶峰，PE低但即将下滑
```

---

#### 3. 医疗保健行业

**制药公司**：
```
□ 专利保护期剩余时间
□ 在研管线数量及阶段
□ 单一产品占比（<30%安全）
□ 药品定价权（医保谈判风险）
```

**CXO公司**：
```
□ 客户集中度（分散为佳）
□ 产能利用率
□ 地缘政治风险（海外收入占比）
□ 订单 backlog
```

**医疗器械**：
```
□ 产品注册证数量
□ 临床试验成功率
□ 医生转换成本
□ 耗材/设备收入比（耗材收入稳定）
```

---

#### 4. 媒体/互联网行业

**关键指标**：
```
□ 用户规模（MAU/DAU）
□ 用户时长
□ 单用户价值（ARPU）
□ 广告加载率
□ 内容成本/收入
```

**竞争优势**：
- 网络效应（微信）
- 规模经济（内容成本分摊）
- 数据优势（个性化推荐）

---

#### 5. 消费品行业

**关键指标**：
```
□ 品牌溢价能力（毛利率 vs 行业平均）
□ 渠道控制力
□ 存货周转（时尚风险）
□ 新品成功率
□ 市场份额稳定性
```

**护城河来源**：
- 品牌（茅台、可口可乐）
- 渠道（伊利、农夫山泉）
- 成本优势（规模效应）

---

#### 6. 金融行业

**银行**：
```
□ 净息差(NIM)
□ 不良贷款率
□ 拨备覆盖率
□ 资本充足率
□ 成本收入比
```

**保险**：
```
□ 综合成本率（<100%承保盈利）
□ 投资收益率
□ 偿付能力充足率
□ 新业务价值(NBV)
```

---

#### 7. 零售行业

**关键指标**：
```
□ 同店销售增长
□ 存货周转天数
□ 坪效（每平方米销售额）
□ 毛利率（差异化能力）
□ 租金/收入比
```

**竞争优势**：
- 成本领先（沃尔玛）
- 差异化（高端百货）
- 便利性（便利店）

### 四、行业对比分析模板

```
行业：__________

1. 增长性评估
   - 历史收入增速：___% (5年CAGR)
   - 未来增长驱动：________________
   - 增长可持续性：高/中/低

2. 盈利性评估
   - 行业平均毛利率：___%
   - 行业平均ROIC：___%
   - 定价权：强/中/弱

3. 竞争格局
   - CR5（前5市占率）：___%
   - 进入壁垒：高/中/低
   - 替代品威胁：高/中/低

4. 周期位置
   - 当前处于：导入/成长/成熟/衰退
   - 投资策略：进攻/防御/回避

5. 风险因素
   - 政策风险：________________
   - 技术变革风险：________________
   - 宏观经济敏感度：高/中/低
```

### 五、行业轮动与宏观匹配

```
经济扩张早期：金融、可选消费
经济扩张中期：科技、工业
经济扩张晚期：能源、材料
经济衰退早期：必需消费、医药
经济衰退晚期：公用事业、REITs
```

## 快速分析工作流

```
Step 1: 确定行业生命周期位置（3分钟）
Step 2: 评估五力模型（5分钟）
Step 3: 应用行业特定指标（5分钟）
Step 4: 对比行业内主要玩家（5分钟）
Step 5: 宏观匹配与风险识别（2分钟）

Total: 20分钟完成行业分析
```
