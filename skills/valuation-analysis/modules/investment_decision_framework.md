# 投资决策整合框架

## 核心框架：《股市真规则》投资决策流程

### 一、投资决策漏斗

```
┌─────────────────────────────────────────────────────┐
│  Step 1: 初筛（5分钟）                                │
│  - 估值检查（PE/PB/FCF Yield）                        │
│  - 财务健康检查（负债率、现金流）                      │
│  - 行业筛选（避开糟糕行业）                           │
└──────────────────┬──────────────────────────────────┘
                   │ 通过 → 进入下一步
                   ▼
┌─────────────────────────────────────────────────────┐
│  Step 2: 深度分析（2-3小时）                          │
│  - 护城河分析（五步检验法）                           │
│  - 财务分析（六步财务检查）                           │
│  - 管理层评估（三维评估）                             │
│  - 行业分析（生命周期、竞争格局）                      │
└──────────────────┬──────────────────────────────────┘
                   │ 通过 → 进入下一步
                   ▼
┌─────────────────────────────────────────────────────┐
│  Step 3: 估值建模（1-2小时）                          │
│  - DCF估值（五步法）                                  │
│  - 同业估值对比                                       │
│  - 安全边际计算                                       │
└──────────────────┬──────────────────────────────────┘
                   │ 通过 → 进入下一步
                   ▼
┌─────────────────────────────────────────────────────┐
│  Step 4: 投资决策                                     │
│  - 综合评分                                           │
│  - 仓位建议                                           │
│  - 买入/卖出价格设定                                  │
└─────────────────────────────────────────────────────┘
```

### 二、综合评分卡

#### 2.1 评分维度与权重

| 维度 | 权重 | 评分标准 | 得分 |
|------|------|---------|------|
| **护城河强度** | 25% | 1-5分 | ___ |
| **财务健康度** | 20% | 1-5分 | ___ |
| **成长性** | 15% | 1-5分 | ___ |
| **盈利能力** | 15% | 1-5分 | ___ |
| **管理层质量** | 15% | 1-5分 | ___ |
| **估值吸引力** | 10% | 1-5分 | ___ |
| **综合评分** | 100% | 加权平均 | ___ |

#### 2.2 评分标准

**5分 = 优秀**：行业顶尖水平，显著超越标准
**4分 = 良好**：高于平均水平，达到投资标准
**3分 = 一般**：平均水平，无明显优势或劣势
**2分 = 较差**：低于平均水平，存在明显问题
**1分 = 糟糕**：行业底部，存在重大风险

### 三、安全边际决策矩阵

#### 3.1 买入决策

| 公司类型 | 安全边际要求 | 综合评分要求 | 仓位建议 |
|---------|-------------|-------------|---------|
| **强护城河稳定公司** | ≥20% | ≥4.0 | 10-15% |
| **一般公司** | ≥30-40% | ≥3.5 | 5-8% |
| **高风险公司** | ≥60% | ≥3.0 | 2-5% |
| **周期性公司** | 周期低谷+30% | ≥3.0 | 5-10% |

#### 3.2 卖出决策

| 卖出信号 | 优先级 | 行动 |
|---------|-------|------|
| 股价 > 内在价值（无安全边际） | 高 | 减仓/卖出 |
| 护城河被侵蚀 | 高 | 重新评估 |
| 管理层诚信问题 | 高 | 立即卖出 |
| 财务健康恶化（3个以上危险信号） | 高 | 减仓/卖出 |
| 发现更好的投资机会 | 中 | 调仓 |
| 仓位过重（单股 >15%） | 中 | 再平衡 |
| 税务优化需求 | 低 | 酌情处理 |

### 四、完整投资检查清单

#### 4.1 买入前检查清单

```
护城河分析
□ 历史盈利能力检查通过（FCF/销售>5%, ROE>15%）
□ 护城河来源清晰（至少1种强护城河）
□ 护城河宽度评估为"宽"或"极宽"
□ 行业竞争结构有利
□ 管理层理性配置资本

财务分析
□ 成长性健康（收入稳定增长）
□ 盈利能力优秀（毛利率、净利润率稳定）
□ 财务健康（负债/权益<1.0, 流动比率>1.5）
□ 无六个危险信号
□ 盈利持续性（10年ROE>15%）
□ 会计质量良好（无操纵迹象）

估值分析
□ DCF估值完成
□ 安全边际达到要求（好公司20%+）
□ 同业估值对比处于合理/低估区间
□ 未来3-5年回报率预期 > 要求回报率

管理层评估
□ 战略能力可信
□ 资本配置记录良好
□ 无诚信红旗信号
□ 股东导向明确

综合判断
□ 综合评分 ≥ 3.5（一般公司）或 ≥ 4.0（好公司）
□ 理解业务并能跟踪关键指标
□ 愿意持有3-5年以上
```

#### 4.2 持有期监控清单

```
季度检查
□ 收入/利润是否符合预期
□ 毛利率/净利润率趋势
□ 经营现金流 vs 净利润
□ 关键运营数据（用户、ARPU等）

年度检查
□ 护城河是否被侵蚀
□ 财务健康是否恶化
□ 管理层是否变化
□ 估值是否脱离安全边际
□ 投资理由是否仍然成立
```

### 六、投资组合管理原则

#### 6.1 仓位管理

```
单一个股上限：15%
单个行业上限：25%
现金储备：10-20%（等待机会）

加仓规则：
- 首次建仓：目标仓位的30-50%
- 下跌10%且有安全边际：加仓至70%
- 下跌20%且基本面未变：加满
- 上涨脱离安全边际：停止加仓
```

#### 6.2 再平衡规则

```
定期再平衡：每年一次
触发再平衡：
- 个股仓位 > 15%
- 行业仓位 > 25%
- 发现明显更好的机会

再平衡方法：
- 减仓高估标的
- 加仓低估标的
- 保持组合整体安全边际
```

### 七、常见错误与规避

| 错误类型 | 具体表现 | 规避方法 |
|---------|---------|---------|
| **估值陷阱** | 低PE但盈利不可持续 | 检查盈利质量和持续性 |
| **价值陷阱** | 低估值但业务衰退 | 确认护城河未被侵蚀 |
| **成长陷阱** | 高成长但无护城河 | 护城河分析先于成长性 |
| **周期陷阱** | 周期顶峰买入 | 识别周期位置 |
| **管理层陷阱** | 忽视管理层质量 | 管理层评估不可少 |
| **过度集中** | 单股仓位过重 | 严格遵守仓位上限 |
| **频繁交易** | 缺乏耐心 | 设定持有期，减少操作 |

## 快速决策工作流

```
完整分析流程（4-5小时）：

Step 1: 初筛（5分钟）
Step 2: 护城河分析（30分钟）
Step 3: 财务分析（25分钟）
Step 4: 行业分析（20分钟）
Step 5: 管理层评估（30分钟）
Step 6: 估值建模（90分钟）
Step 7: 综合评分与决策（20分钟）

简化流程（1小时）- 适用于熟悉的公司：
Step 1: 快速财务健康检查（10分钟）
Step 2: 护城河快速评估（10分钟）
Step 3: 更新DCF估值（30分钟）
Step 4: 决策（10分钟）
```
