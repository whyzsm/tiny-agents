# 镜头运动详细参数与案例

## 推（Dolly In / Push In）

**核心参数：**
- 速度：慢速（5-10cm/s）= 沉重/悬疑；中速（15-25cm/s）= 正常；快速（30cm/s+）= 紧张
- 焦距：广角推 = 透视夸张；长焦推 = 压缩感
- 距离：全身→特写约2-3米，特写→大特写约0.5米

**案例：**
- 《寄生虫》雨中推镜头：从窗外推进室内，增强窒息感
- 《地心引力》推进宇航员：太空中的孤立感

**搭配建议：**
- 推 + 升：史诗感放大
- 推 + 跟：跟随主体同时聚焦
- 推 + 手持：临场 + 紧张

---

## 拉（Dolly Out / Pull Out）

**核心参数：**
- 速度：快速拉出 = 震撼/揭示；慢速拉出 = 沉思/余韵
- 景别跨度：小→大（CU→WS）最常用
- 高度变化：可配合升降

**案例：**
- 《狂飙》大结局拉镜头：从人物拉到空旷街道，孤独感
- 《布达佩斯大饭店》对称拉出：喜剧+荒诞

---

## 摇（Pan）

**核心参数：**
- 角度：小于30° = 小扫视；30°-90° = 中等；90°+ = 大范围扫描
- 速度：快速横扫 = 紧迫；慢速 = 观察
- 跟随速度：与主体速度匹配

**案例：**
- 公路片开篇：车窗外依次掠过的风景
- 侦探场景：镜头从证人扫到嫌疑人

---

## 移（Move / Truck）

**核心参数：**
- 轨道类型：直轨 = 干净利落；弯轨 = 变化感
- 速度：与主体速度相对或相同
- 镜头高度：高轨 = 俯视感；低轨 = 贴近地面

**案例：**
- 《人类之子》长镜头跟随：车内逃亡的沉浸感
- 《俄罗斯方舟》无剪辑长推：穿越博物馆

---

## 跟（Follow / Tracking）

**核心参数：**
- 距离：近跟 = 亲密/紧张；远跟 = 客观/旁观
- 高度：平视 = 平等；仰拍 = 崇拜；俯拍 = 优越感
- 障碍物：穿越障碍增加动感

**案例：**
- 追逐戏：跟跑者背后，汗水飞溅
- 纪录片：跟拍人物日常走路

---

## 升降（Crane / Jib）

**核心参数：**
- 高度差：大升降（1m以上）= 史诗；小升降 = 微妙
- 速度：慢升 = 揭示；快升 = 震撼
- 旋转配合：升降时是否旋转

**案例：**
- 《好家伙》从吧台升到天花板：黑帮的气场
- MV开场：从地面升到城市全景

---

## 俯仰（Tilt）

**核心参数：**
- 角度：小于45° = 审视；45°-90° = 戏剧化
- 速度：快速 = 震惊；慢速 = 打量
- 配合：与人物身高差配合效果最佳

**案例：**
- 仰拍高楼：都市的压迫感
- 从脚到脸扫视：全面审视一个人

---

## 环绕（Orbit / Revolution）

**核心参数：**
- 半径：大半径 = 冷静观察；小半径 = 聚焦紧张
- 速度：匀速 = 稳定；变速 = 情绪波动
- 高度变化：平绕 vs 斜绕

**案例：**
- 《搏击俱乐部》环绕两人对话：对峙感
- 商业广告环绕产品：全方位展示

---

## 旋转（Spin / Roll）

**核心参数：**
- 角度：90°-180° = 轻度眩晕；360° = 强烈失控
- 速度：快速 = 失控；慢速 = 沉思
- 镜头语言：配合剧情高潮使用

**案例：**
- 梦幻/醉酒场景
- 情绪爆发/崩溃时刻

---

## 固定镜头（Static Shot）

**核心参数：**
- 景别决定张力：远景固定 = 环境感；近景固定 = 凝视
- 时长：3秒以下 = 快速过渡；3-8秒 = 正常；8秒+ = 强调/观察
- 内部运动：镜头不动但画面内有人物运动

**案例：**
- 斯坦库·库布里克式固定对话
- 韦斯·安德森对称构图固定镜头

---

## 手持镜头（Handheld）

**核心参数：**
- 抖动程度：轻微 = 真实；明显 = 紧张；剧烈 = 失控
- 呼吸节奏：配合演员呼吸
- 斯坦尼康 = 加固手持，可稳定也可保持轻微晃动

**案例：**
- 《拯救大兵瑞恩》诺曼底登陆：临场感
- 《科洛弗道10号》：密室恐惧

---

## 组合运镜公式

| 公式 | 效果 | 适用场景 |
|------|------|----------|
| 推 + 摇 | 聚焦+扫视 | 发现关键物体 |
| 跟 + 推 | 临场+聚焦 | 追逐+锁定 |
| 环绕 + 升 | 揭示+史诗 | 大场景开场 |
| 拉 + 摇 | 开放+探索 | 场景结束 |
| 固定 + 内部运动 | 稳定张力 | 对话/观察 |
| 手持 + 快速剪辑 | 真实紧张 | 动作/混乱 |
