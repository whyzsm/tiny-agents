# 分镜脚本模板与填写说明

## 基础分镜格式

```
┌─────────────────────────────────────────────────────┐
│ 场景编号: ___  场景名称: ________________________   │
│ 地点: ________________  时间: ___________________  │
├────┬──────┬────────┬──────────┬────────┬───────────┤
│镜号│ 景别 │ 运镜   │ 机位/角度 │ 时长   │ 画面描述   │
├────┼──────┼────────┼──────────┼────────┼───────────┤
│ 1  │      │        │          │        │           │
│ 2  │      │        │          │        │           │
└────┴──────┴────────┴──────────┴────────┴───────────┘
备注：_____________________________________________
```

---

## 字段填写指南

### 景别（按从小到大）
- **ECU** 大特写（眼睛/局部细节）
- **CU** 特写（面部）
- **MS** 中近景（胸口以上）
- **MCU** 中近景（腰部以上）
- **WS** 远景（人物全身+环境）
- **EWS** 极远景（空旷环境）

### 运镜类型
- `固定` 机身不动
- `推` 向前推进
- `拉` 向后拉远
- `摇` 水平旋转
- `移` 横向/纵向移动
- `跟` 跟随主体
- `升降` 上下运动
- `俯仰` 垂直旋转
- `环绕` 圆周运动
- `手持` 随机抖动
- `组合` 两种以上运镜

### 机位/角度
- `平视` 与主体等高
- `仰拍` 低于主体，向上拍
- `俯拍` 高于主体，向下拍
- `侧拍` 90°侧面
- `背面` 从背后拍
- `斜侧` 45°侧面

### 时长参考
- 过渡镜头：1-3秒
- 正常叙事：3-8秒
- 特写强调：1-3秒
- 全景交代：5-10秒
- 情绪镜头：8秒以上

## 完整分镜示例

```
场景: 咖啡馆邂逅
总时长: 约45秒

┌─────────────────────────────────────────────────────┐
│ 场景编号: 03  场景名称: 咖啡馆邂逅                   │
│ 地点: 咖啡馆内  时间: 下午                          │
├────┬──────┬────────┬──────────┬────────┬───────────┤
│ 1  │ EWS  │ 固定   │ 俯拍     │ 4秒    │ 咖啡馆全貌，│
│    │      │        │          │        │ 人来人往   │
├────┼──────┼────────┼──────────┼────────┼───────────┤
│ 2  │ WS   │ 移     │ 平视     │ 5秒    │ 镜头平移，  │
│    │      │        │          │        │ 停在角落   │
├────┼──────┼────────┼──────────┼────────┼───────────┤
│ 3  │ CU   │ 推     │ 平视     │ 3秒    │ 她在看书，  │
│    │      │        │          │        │ 镜头推向书  │
├────┼──────┼────────┼──────────┼────────┼───────────┤
│ 4  │ MCU  │ 固定   │ 侧面     │ 6秒    │ 男子走进    │
│    │      │        │          │        │ 画面，坐下  │
├────┼──────┼────────┼──────────┼────────┼───────────┤
│ 5  │ CU   │ 拉     │ 平视     │ 4秒    │ 她抬头，    │
│    │      │        │          │        │ 镜头缓缓拉远│
└────┴──────┴────────┴──────────┴────────┴───────────┘

备注：镜头1交代环境，镜头2建立空间，镜头3聚焦女主，
     镜头4引入男主，镜头5对视瞬间，全片无台词。
```

## 快速检查清单

在完成分镜后，逐项检查：

- [ ] 每个镜头有明确的叙事目的
- [ ] 景别变化有逻辑（避免跳切）
- [ ] 运镜速度与情绪匹配
- [ ] 起幅和落幅画面稳定
- [ ] 总时长符合实际需求
- [ ] 剪辑节奏有呼吸感（不是一直快）
- [ ] 有动有静，有近有远
